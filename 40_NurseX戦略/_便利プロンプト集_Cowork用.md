## 🗂 Vault管理・ファイル操作

### 新しいファイルを指定フォルダに作る
```
[フォルダ名] に [ファイル名].md を作って。
内容は [テーマ・メモの内容] について。
```

### 今のVaultの構成を確認する
```
Obsidian Vaultの現在のフォルダとファイル一覧を見せて。
```

### ファイルを移動する
```
[元のファイル名] を [移動先フォルダ] に移動して。
```

### 00_受信箱を整理する
```
00_受信箱の中のファイルを一覧で出して、どのフォルダに移すか提案して。
私が確認してからまとめて移動してほしい。
```

## ✍️ note記事・ブログ記事作成

### ドラマ考察記事をゼロから作る
```
ドラマ「[ドラマ名]」の [エピソード・シーン名] を考察したnote記事を書いて。

テーマ軸：[アセスメント思考 / INFJ視点 / 2番手の法則 / キャリア / 自分株式会社] から選んで。
構成：
- 冒頭（読者を引き込むシーン描写）
- 考察本文（3段落）
- 実生活への応用
- 締め（読者への問いかけ）

文体：_文体・口癖メモ.md を参照してこういちの口調で。
```

### メモ書きから記事の骨格を作る
```
以下のメモから note 記事の骨格を作って。

【メモ】
[ここにメモを貼る]

条件：
- タイトル案3つ
- 見出し構成（H2×3〜4）
- 各見出しに2〜3行のメモ書き
- 推奨文字数の目安
```

### タイトル案を出す
```
この記事のタイトル案を5つ出して。

記事テーマ：[テーマ]
ターゲット：看護師・転職・キャリアに悩む30〜50代
条件：クリックしたくなるタイトル。数字・疑問・逆説を使う。
```

### 執筆中ファイルを仕上げる
```
30_記事製造ライン/執筆中/[ファイル名].md を読んで、
このドラフトをnote公開用に仕上げて。

_文体・口癖メモ.md と _コアテーマ軸.md も参照すること。
仕上げたら 30_記事製造ライン/公開済み_note/ に保存して。
```

## 📓 日報・振り返り

### 今日の日報を作る
```
今日の日報を作って。

【今日やったこと・気になったこと】
[ここにメモを貼る]
```

### 日報から来週の戦略を立てる
```
60_日報/ の直近1週間分の日報を読んで、
来週のランクA・B・Cのタスクリストを作って。
Obsidianに貼れるMarkdown形式で出力して。
```

### セッションの成果をまとめる
```
セッション終了。今日の作業まとめをObsidianに保存して。
```

## 🧠 アイデア出し・ブレスト

### 制約付きブレスト（Bコマンド）
```
B [テーマ]
```
> 冒頭に「B」をつけると、極端な制約を自分で設定してエッジの効いたアイデアを量産するモードに入る。

### NurseX記事の横展開ネタを出す
```
_NurseX記事アイデアリスト.md と、最近書いた記事を見て、
まだ書いていない角度・テーマを5つ提案して。
```

### コアテーマ軸 × 最近の出来事でネタ出し
```
以下の出来事を、_コアテーマ軸.md のいずれかと掛け合わせて
note記事のネタになるか考えて。

【出来事・気づき】
[ここに書く]
```

## 🔄 リライト・加筆

### 既存記事をINFJ軸でリライト
```
30_記事製造ライン/執筆中/[ファイル名].md を
INFJ視点・生存戦略テーマで書き直して。
Nijijourney用の画像プロンプトもセットで作って。
```

### 記事に体験談を追加する
```
30_記事製造ライン/執筆中/[ファイル名].md を読んで、
[〇〇のエピソード] を実体験として盛り込んで加筆して。
語尾や文体は _文体・口癖メモ.md に合わせること。
```

### タイトルと冒頭だけを磨く
```
[ファイル名] のタイトルと冒頭100字だけを、
クリックされやすく・続きを読みたくなるように書き直して。
3パターン提案して。
```

## 📊 分析・整理

### プロンプト集に新しいプロンプトを追加する
```
プロンプト集（_プロンプト集.md または _便利プロンプト集_Cowork用.md）の
[カテゴリ名] に以下のプロンプトを追加して。

【プロンプト名】
[名前]

【プロンプト本文】
[内容]
```

### 公開済み記事の棚卸しをする
```
30_記事製造ライン/公開済み_note/ 以下の記事一覧を出して。
タイトル・テーマ・考察 or 体験記の分類で表にまとめて。
```

### 知識ベースを更新する
```
今日学んだことをまとめるから、
20_知識ベース/[フォルダ名]/[ファイル名].md に追記して。

【内容】
[ここに書く]
```
