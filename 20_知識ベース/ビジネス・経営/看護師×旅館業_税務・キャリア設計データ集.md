## 1. 旅館業の収益段階別・最適事業形態

### 所得税率（個人）vs 法人税率の比較表

| 課税所得 | 所得税率 | 住民税 | 合計（限界税率） |
|---------|---------|-------|----------------|
| ～195万円 | 5% | 10% | 約15% |
| 195～330万円 | 10% | 10% | 約20% |
| 330～695万円 | 20% | 10% | 約30% |
| 695～900万円 | 23% | 10% | 約33% |
| 900万円～1,800万円 | 33% | 10% | 約43% |
| 1,800万円～4,000万円 | 40% | 10% | 約50% |
| 4,000万円超 | 45% | 10% | 約55% |

出典：国税庁「No.2260 所得税の税率」

個人事業主の最大税率：**最大60%**（所得税45%＋住民税10%＋個人事業税5%）

### 法人税率（中小企業・資本金1億円以下）

| 課税所得 | 法人税率 |
|---------|---------|
| 800万円以下の部分 | **15%**（軽減税率） |
| 800万円超の部分 | **23.2%** |

実効税率（法人税＋法人住民税＋法人事業税の合計）：**約26～30%前後**

- 法人住民税（均等割）：最低7万円/年（赤字でも必要）
- 法人事業税：400万以下3.5%、400万超800万以下5.3%、800万超7%

出典：freee「法人化のタイミングと目安年収」

### 法人化の分岐点（利益ベースでの正しい判断）

**「年収ではなく課税所得（利益）800万円が分岐点」は正しい。**

| 課税所得 | 個人手取り | 法人化後手取り | 差額 |
|---------|-----------|-------------|-----|
| 500万円 | 約374万円 | 約383万円 | +約9万円（法人有利だが社保増で微差） |
| 800万円 | 約561万円 | 約565万円 | +約4万円（損益分岐点・維持費考慮で実質トントン） |
| 1,200万円 | 約783万円 | 約830万円 | **+約47万円（法人化明確に有利）** |

出典：note「法人化の節税は利益800万円が分岐点」（税理士監修）

**結論：課税所得（旅館業の純利益）が800万円を超えてから法人化検討が現実的。1,000万円超で効果が明確になる。**

### 看護師（給与所得）保有者の特殊事情

看護師として勤務先の厚生年金・健康保険に加入しているサラリーマンが副業（個人事業）を持つ場合：

- **社会保険は勤務先の厚生年金・健保が継続される（国保への変更不要）**
- 副業の事業所得が増えても、追加の社会保険料負担は発生しない（個人事業主として副業の場合）
- ただし年収130万円超で扶養から外れる場合は注意

## 2. マイクロ法人戦略の詳細

### マイクロ法人とは

1人で設立・経営する小規模法人。個人事業主との「二刀流」で節税・社保削減を狙う戦略。

### 社会保険節税の仕組み（看護師×旅館業法人の場合）

**前提：すでに勤務先で厚生年金・健保に加入している看護師の場合**

> **重要：サラリーマン（給与所得者）がマイクロ法人を設立しても、社会保険料の節約はできない。**

理由：
- 勤務先で厚生年金・健保に加入継続
- マイクロ法人でも役員報酬を受け取ると、**法人格でも別途厚生年金・健保への加入義務が発生**
- 結果：二重加入となり保険料が増える

出典：bring-consulting「マイクロ法人はデメリットも多数」

**ただし、旅館業を個人事業（事業所得）として維持しながら、別の事業（情報発信・コンテンツ販売等）でマイクロ法人を設立する場合は、社保節約の効果を得られる可能性がある。**

### 個人事業主として旅館業を続ける場合の社会保険

看護師として勤務先の社保に入りつつ、旅館業は個人事業のまま保つ→**追加の社会保険料は不要**

### マイクロ法人のコスト

**設立費用：**
| 形態 | 設立費用（法定費用のみ） |
|-----|----------------------|
| 株式会社 | 約22万2,000円～ |
| 合同会社（LLC） | 約10万円～ |

**年間維持費（最低ライン）：**
| 項目 | 費用 |
|-----|-----|
| 役員報酬（月4.5万円×12） | 54万円 |
| 会社負担の社会保険料 | 約14万円 |
| 法人住民税（均等割・赤字でも） | 7万円 |
| 税理士費用（決算申告のみ） | 10万円 |
| その他経費 | 約5万円 |
| **合計** | **約90万円** |

税理士顧問契約（月次）を付けると：月1～5万円（年12～60万円）が追加

### マイクロ法人が有効になる損益分岐点

**個人事業主として副業（旅館業）のみの場合：**

- 扶養なし・40歳未満で年間所得**200万円以上**から社会保険料節約効果が維持費を上回り始める（約18万円の節約）
- ただし看護師（給与所得）として既に職場の社保加入済みの場合は上記の計算式が変わる

**看護師+旅館業法人（役員報酬最小化）の最適設計：**

役員報酬の最適設定（東京都基準・2024年度）：
- 役員報酬：月額**11,411円～45,000円**が最適
- 月額45,000円以下 → 年54万円以下 → 給与所得控除55万円の範囲内 → **役員報酬に所得税ゼロ**
- 月額63,000円未満 → 社会保険の1等級（最安）が適用される

出典：協会けんぽ「令和6年3月分からの保険料額表（東京都）」/ 国税庁「No.1410 給与所得控除」

## 3. 副業旅館業の確定申告・税務リスク

### 看護師が副業で旅館業を持つ場合の確定申告上の注意点

1. **旅館業（旅館業法許可）の所得は「事業所得」として申告**
   - 青色申告（65万円控除）の適用が可能
   - 帳簿の記録・保存が必要（7年間）

2. **副業所得20万円超で確定申告必須**
   - 看護師の給与（主所得）＋旅館業事業所得が20万円超 → 確定申告義務

3. **個人事業税（5%）も課税対象**
   - 旅館業は第1種事業として5%（ただし年290万円の控除あり）

### 住民税から副業がバレるリスクと対策

**バレる仕組み：**
- 確定申告後、住民税が合算計算される
- 役所から勤務先に「住民税決定通知書」が届く
- 通知書の「主たる給与以外の合算所得区分」に○が付く → 経理担当者が把握できる

**対策（合法的な方法）：**
- 確定申告書「第二表」の住民税徴収方法欄で**「自分で納付（普通徴収）」を選択**
- 副業が「事業所得」（給与所得ではない）場合は普通徴収が選択可能
- ただし職場に副業禁止規定がある場合は別途確認が必要

> 注意：旅館業（事業所得）は普通徴収を選択できる。看護師副業バイト（給与所得）は原則普通徴収不可。

出典：passmed「看護師の副業がバレるのはどんな時」/ マネーフォワード「副業は住民税でバレる？」

### 旅館業で経費計上できる主要項目

| 経費種別 | 具体的な内容 |
|---------|-----------|
| **修繕費** | 建物・設備・備品の修理・メンテナンス |
| **減価償却費** | 建物・設備・車両の取得費用（年数で償却） |
| **水道光熱費** | 電気・ガス・水道（事業割合で按分） |
| **通信費** | 予約システム・電話・Wi-Fi |
| **旅費交通費（送迎）** | 顧客送迎の車両費・ガソリン代・ETC |
| **広告宣伝費** | OTA手数料・自社サイト制作・チラシ |
| **損害保険料** | 旅館業向け損害保険 |
| **租税公課** | 固定資産税・自動車税（事業用車） |
| **地代家賃** | 施設の土地・建物の賃借料（所有なら計上不可） |
| **消耗品費** | アメニティ・清掃用品・寝具等 |
| **外注費** | 清掃業者・管理会社への委託費 |
| **接待交際費** | 取引先（旅行会社等）との打ち合わせ飲食代 |

> 自宅兼用施設の場合は**按分（使用面積・時間比率）**が必要

## 4. 実例分析：看護師×宿泊業の税務メリット・デメリット

### メリット

1. **損失の通算（赤字の活用）**
   - 旅館業で初年度赤字が出た場合、給与所得と損益通算は不可（事業所得と給与所得の通算は原則不可）
   - ただし青色申告なら**翌年3年間の繰越控除**が可能

2. **設備投資の減価償却**
   - 改装費・空調設備・ベッド等は減価償却で複数年にわたって経費計上可能
   - 中小企業者向け少額減価償却資産の特例：30万円未満は即時全額経費化（青色申告限定）

3. **社会保険の心配が少ない**
   - 看護師として職場の厚生年金・健保に加入済み
   - 個人事業（旅館業）の追加所得に対して社会保険料の追加負担なし

### デメリット・リスク

1. **給与所得との損益通算制限**
   - 副業の事業所得の赤字は、給与所得と相殺できない（原則）
   - 「生計の主たる所得でない」副業が赤字を繰り返すと雑所得に格下げされるリスク

2. **税務調査リスク**
   - 旅館業の経費（プライベートと事業が混在しやすい）は税務調査で指摘されやすい
   - 送迎車、自宅電話、光熱費は事業割合の根拠資料を残すことが重要

3. **社会保険二重加入問題（法人化した場合）**
   - 旅館業を法人化して役員報酬を受け取ると、**看護師の職場＋旅館業法人の二重加入が発生**
   - 保険料が二重にかかり、節税どころか負担増になるケースが多い

## 5. 判断フレームワーク：こういちへの適用

### 現状（個人事業主として旅館業）が有利な条件

- 旅館業の年間純利益が**800万円未満**
- 看護師の給与所得から社保を負担済みで、追加社保なし
- 法人の維持費（年90万円）をカバーできるほどの利益がない段階

### 法人化を検討すべきタイミング

- 旅館業の年間課税所得（純利益）が**800万円を安定的に超えたとき**
- 複数の宿泊施設を運営し、給与所得者ではなく法人として事業拡大するとき
- 家族（妻）を役員・従業員にして所得分散したいとき

### マイクロ法人（旅館業以外の事業）を検討すべきケース

- 情報発信・note有料記事・コンサルティング等、旅館業とは別の収益が発生した場合
- その収益をマイクロ法人に集約し、役員報酬を最小化（月4.5万円）して旅館業収入は個人事業のままにする
- 旅館業収入（個人事業所得）の課税は別枠で最適化する
