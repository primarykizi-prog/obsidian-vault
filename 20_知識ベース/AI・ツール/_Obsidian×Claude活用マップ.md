# 🗺️ Obsidian × Claude　活用マップ

> **一言で言うと：**
> Obsidianは「こういちの頭の中の倉庫」、Claudeは「その倉庫を一緒に整理・加工してくれる相棒」。

---

## ✅ 今できること（全部）

---

### 🎤 1. 音声メモ → 記事の種を作る

**やること**
Google Keepで話す → テキストをそのままClaudeに貼る

**一言テンプレ**
```
今日の音声メモです。
①記事の種を2〜3個抽出して、執筆中ノートのフォーマットで作ってください。
②日報の「記事の種」セクションに入れる一文も作ってください。
[音声メモのテキストを貼る]
```

**Obsidianに残るもの**
- `日報/2026-XX-XX.md`（音声メモ原文＋今日の種）
- `執筆中/○○.md`（記事の下書き）

---

### 💡 2. アイデアのタイトルだけ → 記事の骨格を作る

**やること**
思いついたタイトルや一言をClaudeに投げる

**一言テンプレ**
```
「管理職はどこまで管理するの」という記事を作りたいです。
こういちの_コアテーマ軸と_自分取扱説明書をもとに、
執筆中ノートのフォーマットで骨格を作ってください。
```

**Obsidianに残るもの**
- `執筆中/タイトル.md`（フロントマター・核心・構成・エピソード素材メモつき）

---

### 📚 3. 本・記事を読んだ → 自分の記事に横展開する

**やること**
`📚 インプット・参考資料/`に読書メモを書く → Claudeに貼る

**一言テンプレ**
```
以下の読書メモをもとに、こういちの視点（看護師25年・INFJ）で
派生記事の構成を作ってください。パクリではなく自分の話をする方向で。
[読書メモを貼る]
```

**Obsidianに残るもの**
- `📚 インプット・参考資料/本のタイトル.md`（読書メモ）
- `執筆中/派生記事タイトル.md`（横展開した記事骨格）

---

### 🧠 4. NotebookLMで深掘り・棚卸しする

**やること**
Obsidianのファイル数本をNotebookLMにアップ → プロンプト集のA〜Fを使う

**代表プロンプト（NotebookLMに入力）**
- **思考パターン抽出**：「この人が繰り返し使っている思考の型を3つ抽出してください」
- **記事深掘り**：「この記事に足りない視点・深めるべき部分を3つ指摘してください」
- **つながり発見**：「複数のノートに共通するテーマやパターンはありますか？」
- **読者ペルソナ**：「この記事を読んで最も救われそうな読者像を描いてください」

**詳細**：[[_NotebookLM活用ガイド]] 参照

---

### ✍️ 5. Obsidianの下書き → Cursorで仕上げる

**やること**
`執筆中/○○.md`をCursorで開く → Claudeが書いた骨格をもとに本文を仕上げる

**流れ**
```
Obsidianで骨格作成（Claude）
    ↓
Cursorで開く
    ↓
「以下の骨格をもとに、_文体・口癖メモの文体で本文を仕上げてください」
    ↓
記事完成 → 公開
```

---

### 🔍 6. 自分のOSを更新・確認する

**やること**
Claudeとの会話から出てきた気づき・思考パターンをObsidianに反映する

**更新するファイル**
| ファイル | 更新するタイミング |
|---|---|
| `_自分取扱説明書.md` | 新しい判断基準・行動パターンに気づいたとき |
| `_文体・口癖メモ.md` | 新しい言い回しが出てきたとき |
| `_コアテーマ軸.md` | 書きたいテーマが増えたとき |

**一言テンプレ**
```
今日の会話から、私の思考パターンや口癖で気づいたことがあれば抽出してください。
_自分取扱説明書に追記できる形で教えてください。
```

---

### 💬 7. 制約付きブレストモード（Bコマンド）

**やること**
記事のアイデアや方向性を発散させたいとき、批判なしでブレストする

**使い方**
```
Bコマンド：「看護師が副業するリスクについて」でブレストしてください。
```

- 1回で10〜20のアイデアが出る
- 気に入ったものだけ`Google Keep_ノート記事アイディア.md`に追加
- 詳細は[[_プロンプト集]]参照

---

### 📓 8. 日記を書いた → 明日の戦略をもらう

**やること**
日報を書いた後、その内容をClaudeまたはNotebookLMに貼る

**一言テンプレ**
```
今日の日記です。プロンプト集のDカテゴリのフォーマットで
明日の戦略を出力してください。
[日記全文を貼る]
```

**出力される内容**
- 🎯 **ランクA/B/C**：明日やることを優先順位つきで
- 💊 **一撃の処方箋**：思考の停滞・ボトルネックへの1行アドバイス
- ⚖️ **戦略マトリクス**：やること × なぜ大事かを表で整理
- 🌙 **21:00の鉄則**：デジタルデトックス・翌朝ルーティン
- 💡 **AIフィードバック・資産の種**：将来の知的財産になりそうな気づき

**ポイント**
> 出力には重要な概念が `[[wikilinks]]` 付きで出てくる。
> そのままObsidianに貼るだけで、ノート同士がつながっていく。

**どこに使うか**：`日報/YYYY-MM-DD.md` の末尾に貼り付けるだけ

## 📊 全体フロー（1枚で見る）

```
話す
  └─ Google Keep音声メモ
         ↓（テキストをコピー）
貼る
  └─ Claudeに送る
         ↓
Obsidianに入れる
  ├─ 日報（音声原文＋今日の種）
  └─ 執筆中ノート（記事の骨格）
         ↓
日記フィードバック               ← ★NEW（⑧）
  └─ 日記を貼る → 明日の戦略・資産の種が出力
         ↓
深掘り
  └─ NotebookLM（プロンプトA〜G）
         ↓
仕上げる
  └─ Cursor（本文執筆）
         ↓
公開
  └─ ブログ / note
```
