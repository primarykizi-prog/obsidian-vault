|---|---|

### STEP 2｜管理会社へのアプローチ（並行して動かす）
**なぜ早いか：** 管理会社が決まらないと遠隔運営が始まらない。問い合わせ〜返信〜面談まで1〜2ヶ月かかる。

| | 内容 |
|---|---|
| **担当** | AIチーム（Claude）がメール下書き → こういちさんが送信 |
| **候補** | 民泊管理バンク・LDKプロジェクト・STAYCATION（上位3社） |

| **参考** | [[管理会社比較検討シート]] / [[リベシティ相談テンプレート集]] |
| **状態** | 🟡 メール下書き作成→送信待ち |

---

### STEP 3｜糸魚川病院への打診（看護師採用連携）
**なぜやるか：** 元スタッフとして話せる。病院にとってもWin。通年稼働の柱になる。

| | 内容 |
|---|---|
| **担当** | こういちさん本人（元同僚・師長クラスに直接連絡） |
| **手段** | メール or 電話（元職員として直接でOK） |
| **伝える内容** | 「糸魚川に民泊を開く。移住検討中の看護師の試住先として紹介してほしい」 |
| **ゴール** | 「興味がある」という返事をもらう。正式な覚書は後でいい |
| **状態** | 🟡 打診メールテンプレート作成済み→連絡待ち |

---

| | 内容 |
|---|---|

| **具体的なお願い** | 施設の写真を撮って送ってもらう（まずこれだけ） |
| **目標** | 「やってみる」という言葉をもらう |
| **状態** | 🔴 未着手 |
