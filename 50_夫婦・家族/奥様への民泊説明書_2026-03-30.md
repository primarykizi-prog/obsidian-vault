# 【ご提案】糸魚川の実家（元民宿）を活用した小規模宿泊事業について

## 1. 結論：何をしようとしているのか？

実家を「売却」や「賃貸」に出すのではなく、両親が住んだままの状態で、空き部屋を活用した**「小規模な民泊（家主居住型）」**としてテスト稼働させたいと考えています。

以前はこういった副業を一人で調べて進めようとすると、手間と時間がかかりすぎて現実的ではありませんでした。ただ、今はAIを活用することで、調査・書類作成・集客・やりとりの多くをサポートしてもらえる環境が整っています。「こういちが一人で全部抱え込む」のではなく、**「AIというパートナーと一緒に進める」**という形なので、本業や日々の生活への影響を最小限に抑えながら動けます。

---

## 2. なぜ、この方法を選ぶのか？（3つの理由）

### ① 初期費用とリスクが極めて低い

水回り（風呂・トイレ等）がそのまま使えるため、宿泊業で最もお金がかかる改修工事が不要。必要なのは客室のエアコン買い替えや最低限の備品のみ。市の補助金（最大50万円など）を活用して手出しを最小限に抑える。

補助金の調査や申請書類の下準備も、AIが一緒に調べて整理してくれます。「何が使えるか」「どんな書類が必要か」をAIに確認しながら進められるので、ゼロから一人で調べる必要はありません。

### ② 確実に「勝てる」独自の強みがある

広い駐車スペース（観光バス3台分）があるため、車にバイクを積んでくる「オフロードバイク層」に最適な環境。近くの神社（バイクの聖地）との繋がりを活かし、予約サイトの高い手数料を払わずに直接集客ができる。

集客用の文章やSNS投稿、予約受付の仕組みづくりも、AIがたたき台を作ってくれます。私が「こういう層に向けて発信したい」と伝えるだけで、文章や構成の案を出してもらえるので、センスや経験がなくても形にできます。

### ③ 両親の「生きがい」に繋がる

住み慣れた家から引っ越す（売却する）負担をかけずに済む。重労働はさせず、「管理人」としての適度な役割を持ってもらうことで、日々の張り合いや健康維持（QOL向上）に貢献できる。

## 3. 懸念されるリスクと対策

| 懸念点 | 対策 |
|---|---|
| 本業（看護）や今の生活に支障が出ないか | 私の役割は「遠隔でのネット予約管理」と「集客の仕組みづくり」に特化。現地対応は両親が行うため、頻繁に糸魚川へ通う必要はない。さらに、予約管理の仕組み・集客文章・メール返信の下書きはAIがサポートしてくれるため、実質的な手間はかなり限られる |
| 両親に重労働やトラブル対応の負担がかからないか | 両親の役割は「チェックイン時の軽い挨拶」と「緊急時の備え（家にいること）」のみ。清掃が負担になればシルバー人材センターに外注する |
| 全くお客さんが来なかったら借金を抱えないか | 年間180日の「民泊」枠でまずは小さくテスト。初期投資はエアコン代等のみのため、うまくいかなくても大きな負債は残らない |
| 市役所や管理会社とのやりとりが難しくないか | メールや問い合わせの下書きはAIに作ってもらえる。「こういう件で相談したい」と伝えるだけで、丁寧な文章にまとめてくれるので、手続きのハードルが大きく下がっている |

## 4. AIが入ることで変わること（新しい強み）

「副業を始める」というと、一人で膨大な情報を調べて、書類を書いて、文章を考えて…というイメージがあるかもしれません。ただ、今はそれがかなり変わっています。

- **補助金の調査・申請書類の下準備** → AIが一緒に調べて、必要な内容を整理してくれる
- **予約管理の仕組みづくり・集客文章の作成** → AIがたたき台を出してくれるので、私はチェックと判断だけ
- **市役所や管理会社へのメール対応** → AIが下書きを作ってくれるので、送る前に確認するだけでOK
- **情報収集・選択肢の比較** → AIに「この場合どっちがいい？」と聞けば、メリット・デメリットをまとめてくれる

つまり、「こういちが全部一人でやらなければならない仕事」の量が、以前と比べて格段に少なくなっています。**AIというパートナーと二人三脚で進められる**ので、本業をしながらでも現実的に動けると判断しました。
