## 🌱 テンプレートZ｜【初回投稿用】はじめての相談

**使う場面：** リベシティに初めて投稿するとき。状況を一通り説明して、広く知見を集めたいとき

```
はじめまして。初めて相談させていただきます。
看護師として勤務しながら、実家の物件活用を検討しています。
同じような経験をされた方がいればぜひ教えていただけると幸いです。

━━━━━━━━━━━━━━━━
▼物件の概要
━━━━━━━━━━━━━━━━
・所在地：新潟県の日本海沿いの地方都市（過疎化進行エリア・緊急加速化地域）
・立地：スキー場から車で約1分（海と山が同時に楽しめる珍しいロケーション）
・建物：2棟（本館＋離れ）、収容人数約50人
・駐車場：観光バス3台分
・元の用途：民宿（現在は休眠状態）
・旅館業：廃業済み
・借金：なし
・両親が現地在住・比較的元気（現地サポート役として期待できる）
・自分は車で1時間ほどの距離に在住・本業あり・現地常駐は不可

━━━━━━━━━━━━━━━━
▼現在の状況
━━━━━━━━━━━━━━━━
先日、地元の市役所（空き家バンク担当）に問い合わせたところ、以下のことがわかりました。

・市役所が案内できる補助金は今回のケースではない
・宿泊業の手続きは保健所が窓口
・開業支援の相談は商工会議所へ

旅館業は廃業済みのため、宿泊事業を始める場合は以下の2択になります。
①旅館業法（簡易宿所）として新規申請：日数制限なし・保健所経由・設備基準あり
②民泊新法（住宅宿泊事業法）で届出：年間180日制限あり・手続きは比較的簡単

━━━━━━━━━━━━━━━━
▼やりたいこと
━━━━━━━━━━━━━━━━
・まず「離れだけ」を先行で民泊として活用したい（両親は本館在住のまま）
・管理会社に委託して、自分は遠隔で予約管理や集客の仕組みだけ担当したい
・将来的には老後の移住先候補としても温存しておきたい
・できるだけ初期投資を抑えながらスモールスタートしたい

━━━━━━━━━━━━━━━━
▼教えていただきたいこと
━━━━━━━━━━━━━━━━
①旅館業法（簡易宿所）の新規申請と民泊新法、どちらを選ぶべきでしょうか？
　→過疎エリアで年間稼働を考えると旅館業法の方が有利と思いますが、申請の難易度・費用が気になっています

②地方の元民宿を遠隔で民泊管理している方はいますか？
　→管理会社選びのポイントや失敗談があれば聞かせてください

③過疎地域（緊急加速化地域）の空き家・民泊活用で使える補助金・支援制度をご存知の方はいますか？
　→市役所では案内できないと言われたのですが、他のルートで使える補助金があれば知りたいです

知識ゼロからのスタートですが、コツコツ勉強中です。
経験者の方やご存知の方がいれば、一言でも教えていただけると大変助かります🙏
```

## テンプレートA｜民泊・管理会社について聞く

**使う場面：** 民泊経験者・不動産詳しい人・管理会社を使ったことある人に聞くとき

```
【相談】地方の元民宿を遠隔で民泊運用したい

▼物件の概要
新潟県糸魚川市のスキー場（シーサイドバレー）車1分の元民宿です。
2棟・50人収容・バス3台分の駐車場・借金なし・両親が現地在住でサポート可能。
旅館業許可は失効済みのため、簡易宿所許可の再取得が必要な状況です。

▼やりたいこと
・離れだけ先行で民泊としてスタートしたい
・自分は遠方（看護師勤務中）なので、管理会社に委託して遠隔で運用したい
・緊急加速化地域なので補助金も活用できれば

▼聞きたいこと
①地方物件に対応している信頼できる民泊管理会社を知っていますか？
②管理会社の手数料・初期費用の相場感を教えてほしいです
③旅館業許可の再取得、自分でできますか？行政書士に頼むべき？

同じような経験された方、ぜひコメントください🙏
```

## テンプレートB｜補助金について聞く

**使う場面：** 行政支援・補助金に詳しい人、地方移住・空き家活用の経験者に聞くとき

```
【相談】緊急加速化地域の空き家活用補助金について教えてください

新潟県糸魚川市（緊急加速化地域）に元民宿物件があります。
遠隔で民泊運用を始めるにあたり、使える補助金を探しています。

▼物件の状況
・2棟・50人収容・旅館業許可失効済み・借金なし
・エアコン・トイレ整備すれば使用可能な状態

▼知りたいこと
①糸魚川市の空き家活用補助金・移住定住支援補助金の実態
②緊急加速化地域ならではの上乗せ補助はあるか
③補助金申請のタイミング（着工前が必須？）と手続きの流れ

行政書士に相談する前に、実際に申請した経験のある方の話を聞けると助かります🙏
```

## テンプレートC｜旅館業許可・法的手続きについて聞く

**使う場面：** 宿泊業・民泊の法律に詳しい人、行政書士・司法書士に聞くとき

```
【相談】旅館業許可（簡易宿所）の再取得について教えてください

元民宿（旅館業許可あり→現在失効）の物件を民泊として再稼働させたいです。

▼状況
・新潟県糸魚川市・元民宿・許可失効済み
・民泊新法（180日制限）ではなく、旅館業法で通年営業したい
・消防設備は既存のものがある（確認は必要）

▼聞きたいこと
①旅館業許可（簡易宿所）の再取得は自分でできますか？
②行政書士に頼む場合の費用相場は？
③許可取得までの期間の目安は？
④元民宿であることは、審査で有利になりますか？

経験者の方、ぜひ教えてください🙏
```

## テンプレートD｜管理会社に問い合わせるメール文

**使う場面：** 民泊管理会社に初めてコンタクトするとき

```
件名：元民宿物件の民泊委託運用についてご相談

はじめまして。新潟県糸魚川市に元民宿物件を所有している〇〇と申します。

物件の概要は以下の通りです。

・所在地：新潟県糸魚川市
・最寄り施設：シーサイドバレー糸魚川スキー場（車約1分）
・建物：2棟（本館・離れ）
・収容人数：約50人
・駐車場：観光バス3台分
・現状：エアコン・トイレを整備すれば使用可能
・旅館業許可：失効済み（再取得予定）
・緊急加速化地域に指定されています

私自身は遠方に在住しており、現地常駐が難しい状況です。
両親が現地に住んでおりサポートは可能ですが、運営の主体は管理会社に委託したいと考えています。

つきましては、以下についてご確認させてください。

①このような物件・エリアは対応可能でしょうか
②管理手数料・初期費用の概算を教えていただけますか
③旅館業許可の取得サポートはしていただけますか

ご確認のほど、よろしくお願いいたします。

〇〇（連絡先：〇〇）
```

## テンプレートE｜親（現地サポーター）に説明するとき

**使う場面：** お父さん・お母さんに「民泊のサポートをお願いしたい」と伝えるとき

```
【お父さん・お母さんへ相談したいこと】

離れを使って民泊をやってみたいと思ってる。
自分は遠くにいるから、管理は会社に任せるつもりで、
お父さん・お母さんには「現地サポート役」をお願いできたらと思ってる。

具体的にお願いしたいこと：
・お客さんが来たとき、鍵の受け渡しをしてほしい
・チェックアウト後に「掃除業者がちゃんとやったか」の確認
・何かトラブルがあったとき、最初の電話を受けてほしい（対応は管理会社がやる）

負担は少なくなるよう仕組みを作るから、まず話だけ聞いてほしい。
```
