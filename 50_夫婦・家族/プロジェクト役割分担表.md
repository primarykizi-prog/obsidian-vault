---

## 👤 チームメンバーと役割一覧

---

### 1️⃣ こういち（オーナー・意思決定者）

| 役割 | 具体的なこと |
|---|---|
| **最終意思決定** | 許可申請・契約・料金設定・方針変更などすべての決定 |
| **月次レビュー** | 収益・稼働率・ゲスト評価の確認（月1〜2時間） |
| **資金管理** | 初期投資の判断・補助金申請の最終確認 |
| **関係構築** | 住職・管理会社・両親との関係性を維持する |
| **戦略判断** | AIチームからの提案に対してOK/NGを出す |

**こういちさんがやらなくていいこと：** 日々の予約対応・清掃確認・ゲストメッセージ・経理入力

---

### 2️⃣ お父さん・お母さん（現地サポーター）

| 役割 | 具体的なこと |
|---|---|
| **チェックイン対応** | 鍵の受け渡し・簡単な施設案内 |
| **清掃確認** | 清掃会社が入ったあとの最終チェック |
| **緊急対応** | 設備トラブル時の一次対応・業者への連絡 |
| **施設見回り** | 空き期間中の定期確認（週1〜2回） |
| **現地の目** | 駐車場・設備の状態を写真でこういちに報告 |

**ポイント：** 「手伝い」ではなく、このプロジェクトの**現地責任者という役割**を持ってもらう。
やりがいと感謝を定期的に伝えることが大切。

---

### 3️⃣ 奥さん（相談パートナー・家庭内承認者）

| 役割 | 具体的なこと |
|---|---|
| **意思決定の壁打ち** | 大きな判断前に「これどう思う？」と相談できる存在 |
| **家計への影響確認** | 初期投資・ランニングコストが家計に無理がないか確認 |
| **情報共有** | プロジェクトの進捗を定期的に共有して「知らなかった」をなくす |
| **将来の可能性** | 糸魚川移住・老後の選択肢として一緒に考える |

**ポイント：** 「巻き込む」ではなく、「一緒に考えてほしい」という姿勢で。
重要な節目（開業前・初月収益確定時など）に必ず話す機会を作る。

---

### 4️⃣ 住職・風巻神社（集客パートナー）

| 役割 | 具体的なこと |
|---|---|
| **口コミ・紹介** | 全国から来るライダーへ直接「いい宿があるよ」と伝えてもらう |
| **チラシ設置** | 神社内にチラシを常設してもらう |
| **市場情報の提供** | イベントの日程・参加人数・来場エリアを教えてもらう |
| **権威づけ** | 「風巻神社推薦の宿泊施設」という看板になってもらう |

**ポイント：** 住職にとっても「参拝者が快適に泊まれる場所ができた」はメリット。
Win-Winの関係として丁寧に育てる。

---

### 5️⃣ 民泊管理会社（運営委託先）

| 役割 | 具体的なこと |
|---|---|
| **予約管理** | Airbnb・楽天STAY等のプラットフォーム管理 |
| **ゲスト対応** | 問い合わせ・チェックイン案内・トラブル対応 |
| **清掃手配** | 清掃会社の手配・品質管理 |
| **月次報告** | 稼働率・売上・ゲストレビューをこういちに報告 |
| **緊急対応** | 夜間・休日のトラブルにも一次対応 |

**現状：** 候補5社を比較検討中 → [[管理会社比較検討シート]]
**選定基準：** 地方物件に対応できること・遠隔オーナーに慣れていること

---

### 6️⃣ AIエージェンシーチーム（Claudeが担当）

各エージェントの役割は以下のとおり。詳細は → [[民泊事業ロードマップ_AIチーム設計]]

| エージェント | 主な担当 |
|---|---|
| **📅 予約管理エージェント** | カレンダー管理・空き状況確認・ダブルブッキング防止 |
| **💬 ゲスト対応エージェント** | ウェルカムメッセージ・FAQ回答・レビュー返信の下書き |
| **💰 財務管理エージェント** | 売上・費用の記録・月次レポート作成・収益シミュレーション更新 |
| **📣 マーケティングエージェント** | SNS投稿案・チラシ更新・季節ごとの打ち出し変更 |
| **🔧 現地連携エージェント** | 両親への連絡テンプレート・管理会社への報告確認・備品リスト管理 |

**AIチームができないこと（人間が必要なこと）：**
- 現地での物理的な対応（両親・管理会社が担当）
- 法的契約・申請への署名（こういちさん本人が担当）
- 感情的な関係構築（住職・両親との信頼関係）
