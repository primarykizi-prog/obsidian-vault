作成日：2026-03-29
**最終更新：2026-04-02（タイミング変更・許可見通し後に実施）**

> ⚠️ **実施タイミング変更**：保健所（志賀様）から旅館業許可の見通しが立った段階で訪問する。
> 許可も下りていない段階で「始めます」と言うより、「こういう形で許可が取れそうなので進めたい」という確信を持ってから挨拶する方が誠実かつ説得力がある。
関連：[[差別化戦略_ライダーズベース構想]] / [[ライダーズベース_チラシ案]]

---

## 🎯 この挨拶の目的

1. 正式にチラシ設置の許可をいただく（すでに口頭でOKをもらっているが、実物を見せて確認）
2. イベントの規模感・頻度を直接聞く（市場調査の補完）
3. 住職に「応援者」になってもらう関係を作る

---

## 📋 挨拶前に準備するもの

- [ ] **チラシ（仮版）**を印刷して持参する → [[ライダーズベース_チラシ案]] のHTML版を印刷
- [ ] **物件ワンペーパー**（A4・1枚）を印刷して持参する → 物件概要を口頭で説明するときに使う
- [ ] 手土産（地元のお菓子など、軽いもので十分）
- [ ] メモ帳・ペン（住職から聞いた情報を記録する）

## 🗣️ 挨拶の流れ（トーク例）

### ①自己紹介（1分）
```
「はじめまして（お久しぶりです）。こういちと申します。
糸魚川市の実家に元民宿の物件があり、
ライダーの方の宿泊拠点として活用したいと考えています。
以前、チラシの件でご相談させていただいた者です。」
```

### ②物件の説明（2分）
```
「シーサイドバレーから車で1分のところに、
2棟・50人収容の元民宿があります。
もともと民宿として設計された建物なので、設備もしっかりしています。
旅館業の許可申請を保健所と進めていまして、
365日いつでも受け入れられる施設として再開させたいと思っています。
大きな駐車場もあるので、トランポやバンごと来ていただける施設にしたいと思っています。
室内宿泊のほか、テント泊・車中泊＋トイレ・シャワー利用という形も考えています。」
```

### ③お願い（1分）
```
「全国からライダーの方が集まるこちらの神社で、
チラシを置かせていただけると大変ありがたいです。
まだ仮のチラシですが、こういったイメージで考えています。」
（→ チラシを見せる）
```

### ④ヒアリング（5〜10分・ここが一番大事）
```
・「年間でどのくらいの頻度でイベントがありますか？」
・「1回のイベントで何人くらい来られますか？」
・「遠方から来る方が多いですか？（どのあたりから？）」
・「宿泊先を探している方の声はよく聞きますか？」
・「チラシ以外に、ライダーの方に情報を届ける方法はありますか？（SNS・掲示板など）」
```

### ⑤締め（1分）
```
「本日は貴重なお時間をいただきありがとうございました。
施設の準備が整いましたら、正式なチラシを改めてお持ちします。
引き続きよろしくお願いいたします。」
```
