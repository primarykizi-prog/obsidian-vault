# ✅ MIGAKI BASE 家族すり合わせチェックリスト

> 夫婦と両親、それぞれが「何を望んでいるか」「何が不安か」を洗い出して、
> 全員が納得できる着地点を見つけるためのチェックリスト。
> **こういち夫婦が主導して発信する。両親は設計を知らないから。**

---

## 🎯 前提：こういちの方針

- **スモールスタート**。負荷をかけない
- **需要があるかどうか**で最終判断する（今井さん資料・文左衛門データで検証中）
- 補助金があればベスト、**なくても動ける設計（プランB）**にする
- 何もなくて暗い未来は嫌。**目標がある状態**を作る

---

## 👫 夫婦のすり合わせ（こういち × 奥さん）

### こういちが望んでいること
- [ ] 確認済み：実家を活かして収益化したい
- [ ] 確認済み：看護師を続けながら遠隔で回したい
- [ ] 確認済み：両親の老後の居場所・やりがいを作りたい
- [ ] 確認済み：補助金で家を直せるチャンスを逃したくない
- [ ] 確認済み：将来的に糸魚川との2拠点ライフの可能性を残したい

### 奥さんが望んでいること（確認が必要な項目）
- [ ] 聞いた：家計へのリスクはどこまで許容できるか（上限金額）
- [ ] 聞いた：奥さん自身がどの程度関わりたいか／関わりたくないか
- [ ] 聞いた：糸魚川への訪問頻度に対する希望（月1？年数回？）
- [ ] 聞いた：奥さんが「これがあるならやってもいい」と思う条件
- [ ] 聞いた：奥さんが「これだけは嫌」と思うこと
- [ ] 聞いた：将来の2拠点ライフに対する温度感
- [ ] 聞いた：こういちの時間がどれくらい取られるなら許容範囲か

### 夫婦で決めるべきこと
- [ ] 決めた：初期投資の上限額（「ここまでなら出す」のライン）
- [ ] 決めた：撤退基準（「〇ヶ月やって月〇万円以下なら止める」等）
- [ ] 決めた：収益の使い道（家計に入れる？事業に再投資？貯蓄？）
- [ ] 決めた：奥さんの役割の範囲（CFO的確認だけ？清掃も？）
- [ ] 決めた：両親への伝え方とタイミング（夫婦で揃えてから）
- [ ] 決めた：「成功」の定義（月〇万円？家が綺麗になること？両親の生きがい？）

## 👴👵 両親とのすり合わせ（こういち夫婦 → お父さん・お母さん）

> **両親は設計を知らない。こういち夫婦が噛み砕いて伝える。**
> 4/19の帰省時が最初のすり合わせチャンス。

### 両親が望んでいること（聞き出す項目）
- [ ] 聞いた：家をどうしたいか（このまま？誰かに使ってほしい？）
- [ ] 聞いた：庭や鯉への思い入れ（手放したくないもの）
- [ ] 聞いた：知らない人が家に来ることへの気持ち（抵抗感の有無）
- [ ] 聞いた：自分たちの体力・健康状態への不安
- [ ] 聞いた：冬の雪下ろし・管理の負担についての本音
- [ ] 聞いた：お金の心配（修繕費を出すことへの懸念）
- [ ] 聞いた：「こういちが何かやりたいらしい」への率直な気持ち

### 両親に伝えること
- [ ] 伝えた：「家を売る話ではない。活かす話」
- [ ] 伝えた：「お金は補助金＋こういちが出す。両親の持ち出しはない」
- [ ] 伝えた：「やることは今と変わらない程度（庭と鯉の世話＋見回り）」
- [ ] 伝えた：「知らない人が来るが、プライバシーは守る設計」
- [ ] 伝えた：「家が綺麗になる。屋根も直せるかもしれない」
- [ ] 伝えた：「お父さんの庭の物語が、この事業の一番の武器」
- [ ] 伝えた：「無理はさせない。嫌ならやめる」

### 両親と決めるべきこと
- [ ] 決めた：使っていい部屋・使ってほしくない部屋の線引き
- [ ] 決めた：ゲストと両親の生活動線の棲み分け
- [ ] 決めた：報酬の形（月額固定？歩合？お小遣い的？）
- [ ] 決めた：「やっぱり嫌だ」と言える撤退条件
- [ ] 決めた：お父さんのインタビュー（不動心の庭の話を聞かせてもらう）への了承

## 🎯 全員の共通目標（ここが揃えば動ける）

> **「何もなくて暗い未来」じゃなく、「目標がある状態」を作る。**

### 目標候補（夫婦で話して選ぶ）

**短期（2026年内）**
- [ ] 家が綺麗になっている（エアコン・トイレ・キッチンが直っている）
- [ ] 最初のゲストを迎えている
- [ ] 月〇万円の収益が出ている

**中期（2027〜2028年）**
- [ ] 安定的に月〇万円が入る仕組みができている
- [ ] 両親が「やってよかった」と言っている
- [ ] 糸魚川に家族で泊まりに行ける状態になっている

**長期（2030年〜）**
- [ ] 2拠点ライフの選択肢が現実的になっている
- [ ] MIGAKI BASEが地域に認知されている
- [ ] 看護師を辞めたくなったら辞められる経済基盤がある

## 💡 すり合わせのコツ（こういちメモ）

- **「決めた」ではなく「調べている」から入る**（4/7 今井さんメールの返信で実践済み）
- 奥さんには**数字とデータで説明**する。感情だけだと不安にさせる
- 両親には**具体的なイメージ**で伝える。「エアコンが新しくなる」「トイレが綺麗になる」
- 全員に**撤退ラインを先に見せる**。「ダメならやめられる」が安心感になる
- **目標は「数字」と「気持ち」の両方**を設定する
