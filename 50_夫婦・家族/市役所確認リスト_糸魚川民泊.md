# 糸魚川市役所 確認リスト（民泊・補助金）

市役所への事前相談で確認すべき質問をまとめたリスト。
**窓口：商工観光課 または 地方創生推進課**が最初の相談先。

---

## 📋 確認すること（質問リスト）

### 【届出・許可について】

- [ ] 元民宿（旅館業許可あり）の場合、旧許可は現在も有効か？失効していたら再取得が必要か？
- [ ] 家主居住型民泊（民泊新法・住宅宿泊事業法）の届出手順を教えてほしい
- [ ] 民泊新法（180日制限）と旅館業法（簡易宿所・日数制限なし）、どちらが今回に向いているか？
- [ ] 消防設備（火災警報器・消火器など）は既存のもので問題ないか確認が必要か？
- [ ] 近隣住民への説明義務はあるか？どの範囲まで？
- [ ] 「緊急加速化地域」指定は民泊開業に何か特別な条件や優遇があるか？

### 【補助金・助成金について】

- [ ] 空き家・空き部屋を宿泊施設として活用する場合に使える補助金はあるか？
- [ ] 補助金の上限額・補助率・対象経費（エアコン・備品・改修など）はどこまでか？
- [ ] 申請タイミングと締め切りはいつか？先に工事してしまうと補助が出なくなるか？
- [ ] 「移住・定住支援」や「観光振興」の枠でも使える制度はあるか？
- [ ] 新潟県レベルの補助金と市レベルの補助金、両方使える場合があるか？

### 【運営について】

- [ ] 管理業者（民泊管理会社）への委託は義務か任意か？
- [ ] 両親が現地対応（チェックイン・緊急対応）をする場合、資格や登録は必要か？
- [ ] 外国人旅行者を受け入れる場合に必要な追加手続きはあるか？
