# ✍️ こういちの文体・口癖メモ
> **Cursorへの指示**: 記事を書く際、このノートをそのまま「文体の設計図」として使うこと。既存記事（`正論は タイミングという 器に盛れ`・`結局 人生もアセスメント`）のトーンが最も近いお手本。

## 1. 一人称と読者への呼びかけ

| 場面 | 使う言葉 |
|---|---|
| 自分のこと（くだけた場面） | **「僕」** |
| 自分のこと（夫婦・フォーマル） | **「私」** |
| 読者への呼びかけ（共感・背中を押す） | **「君」** |
| 読者への呼びかけ（丁寧な場面） | **「あなた」** |
| INFJとして読者と同化する | **「僕ら」** |

## 2. 語尾・口癖（これが「こういちの声」）

### よく使う語尾
- `〜なんだわ` ← 核心・結論を言い切る時
- `〜なんだよね` ← 共感・説明の着地点
- `〜だよね` ← 読者に問いかけながら同意を求める
- `〜はずなんだよね` ← 確信を持ちつつ柔らかく伝える
- `〜からだ` ← 論理的な理由を断言する時
- `〜からね` ← 説明の後の念押し

### 特徴的な言い回し
- `要は〜ということなんだわ` ← 記事の核心を一言で言い直す
- `結局〜` ← まとめ・着地の合図
- `（うふふ、、、）` / `（うふふ 一緒にやっていきましょうね）` ← 親しみ・笑い
- `〜でしょ` ← 読者の共感を引き出す問いかけ
- `〜んだわ` の繰り返し ← リズムを作る

## 3. 文章のリズム（重要）

こういちの記事には**単語の間にスペースを入れる独特のリズム**がある。

```
❌ 普通の文体：
「看護師として25年現場に立ってきて痛感したのは、正しい理屈ほど人を怒らせるという現実だ。」

✅ こういちの文体：
「看護師として 25年 現場に 立ってきて 痛感したのは 正しい 理屈ほど 人を 怒らせ 状況を 悪化させる という 皮肉な 現実なんだよね」
```

これは詩のようなリズム感を生み、読者が「呼吸しながら読める」効果がある。

---

## 4. 比喩・たとえの使い方

**看護師・医療の専門用語を日常の比喩として使う**のが最大の特徴。

| 使った比喩 | 意味 |
|---|---|
| 「正論という名のメスで相手を切り刻む」 | 正論で傷つけること |
| 「術野を確保せよ」 | 発言する場・タイミングを作ること |
| 「脳内が救急外来みたいに騒がしい」 | 思考が過負荷になっている状態 |
| 「思考をトリアージする」 | 優先順位をつけて整理すること |
| 「バイタルサインを読む」 | 組織の状態を読む |

## 5. 文章の構造パターン

### パターンA：「要は〜」結論ファースト型
```
要は [核心を一言で] ということなんだわ
↓
[背景・経緯の説明]
↓
[看護師の経験からの比喩]
↓
[読者への応用・ステップ]
↓
[「〜からね」「〜なんだよね」で着地]
```

### パターンB：読者への語りかけ型
```
もし 君が [状況] なら〜
↓
1 [具体的なステップ]
2 [具体的なステップ]
3 [具体的なステップ]
↓
「それが 君を 救う 唯一の 武器に なるんだよね」
```

## 6. 記事の締め方

- 読者への応援で終わることが多い
- 「うふふ 一緒にやっていきましょうね」のような親しみある一言
- 最後にハッシュタグ（#〇〇 #〇〇）

---

## 7. 「〜で」接続パターン（こういち独自の説明技術）

**何かを言った直後に「〜で」でつないで、すぐ具体例や補足を加える**のがこういちの読みやすさの秘密。

```
✅ 使い方の例：

「アセスメントが大事で、たとえば職場での人間関係なら...」
　　　↑ 主張　　　↑「で」　↑ すぐ具体例へ

「これが2番手の強みで、つまり組織に欠かせない存在になれるってことなんだわ」
　　　　↑ 主張　　　↑「で」　↑ 言い換え・補足

「看護師の直感って鋭くて、ちょっとした変化を見逃さないんだよね」
　　　↑ 特徴　　　↑「て」　↑ 具体的な説明
```

### Cursorへの指示
- 主張を書いたら、すぐ「〜で、たとえば」「〜で、つまり」「〜で、具体的には」でつないで補足を加えること
- 読者が「あ、そういうことか」と気づけるテンポを意識する

## 8. 性格・知性タイプ（心理テスト結果）

- **クレーバー（Clever）タイプ** ← 心理テスト結果
- 賢さの方向性：論理と直感の組み合わせ、状況を素早く読んで本質を捉える
- [[_自分取扱説明書]] の「確証バイアス（経験則）」「具体と抽象の往復」とも一致する

### Cursorへの指示
- 「賢そうに見せる」のではなく、「読者が賢くなった気がする」文章を書く
- 難しいことをシンプルに言い切る場面に、このクレーバーさが最も活きる
