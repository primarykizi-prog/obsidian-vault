## 📝 使い方（スキル向けガイド）

1. 記事を書くとき、本文のテーマに近い「テーマタグ」を持つ記事を2〜3本探す
2. 本文末尾に「関連記事」セクションとして以下の形式で挿入する：

```
---
### 📚 関連記事
- [タイトル（短縮）](URL)
- [タイトル（短縮）](URL)
```

3. テーマタグの一致度が高い順に選ぶ。同テーマが3本以上あれば直近の記事を優先する
4. **ドラマ考察記事** → 同じドラマの別記事＋テーマが近い他ドラマ記事を混ぜる
5. **体験エッセイ** → 同カテゴリ＋テーマが近いドラマ考察も1本入れてよい
