## 🔄 自律起動ルール（セッション開始時・必ず実行）

> こういちの指示を待たない。毎セッション開始時に以下を順番にチェックして自律的に動く。

### ① Cronスケジュールの再設定
CronListで確認し、設定がなければ以下を即座に再設定する：
- 月・水・金 8:07 → `digest`
- 月曜 9:03 → `e`（editorial）
- 日曜 21:13 → `weekly`

### ② digestの鮮度チェック
`10_自分OS/_claudian-state.md` の「最終digest」日付を確認する。
- **3日以上経過** → 自動で`digest`実行（こういちへの報告付き）
- 3日以内 → スキップ

### ③ アクションメモのチェック
`10_自分OS/_action-queue.md` を読む。
- **未処理メモあり** → 自動で`a`（reflector-agent）を起動
- なし → スキップ

### ④ 編集判断の鮮度チェック
`10_自分OS/_claudian-state.md` の「最終editorial」日付を確認する。
- **7日以上経過** → 自動で`e`（editorial-agent）を起動
- 7日以内 → スキップ

### ⑤ 会話中のsecretary自動発動ルール
以下のいずれかが起きたら、こういちの指示を待たずにsecretary-agentを起動する：
- アイデア・ビジョン・事業構想が大量に出てきたとき
- 重要な決定事項・方向性が確定したとき
- 新しい事業アイデアや課題が言語化されたとき
→ 「今の内容を保存します」と宣言してから即起動する

## 🔁 CoworkセッションとClaudianの連携ルール（2026-04-02 追加）

**Coworkセッション終了時のフロー**：
「session-end」は使わない。「**日報を作って**」がClaudianへの引き継ぎになる。
日報にCoworkで決めたこと・作ったファイル・得た数値を含めて記録し、Claudianが次回起動時に60_日報を参照して状態を把握する。

## 📋 日報フォーマット拡張ルール（2026-04-02 追加）

`daily-report`スキルで日報を作成する際、`⚖️ 戦略マトリクス`の直前に必ず以下のセクションを追加すること：

```
🏆 今日の報酬チェック（やり切ったら何が得られる？）

ランクAタスクを達成すると → 「（具体的な成果・状態・感情を1文で）」
```

**目的**：報酬感覚プランニングの「重要度チェック」をデイリーに実装。「なぜやるか」を毎日1行書くことで不毛タスクエラーを排除する。
**参照**：[[20_知識ベース/YouTube学習/2026-04-02_報酬感覚プランニング×Obsidianマルチタイムフレーム目標管理]]

## ✍️ 文体ルール（記事執筆時）

- 一人称：「私」、読者：「あなた」
- 短い文を積み重ねる。スペース不使用
- 語尾：`〜だった`（体験）`〜だ`（断言）`〜からだ`（理由）
- **禁止**：`うふふ` / 単語間スペース / `僕・君` / AIっぽい断定
