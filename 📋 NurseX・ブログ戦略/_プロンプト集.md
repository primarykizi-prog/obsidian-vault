## A. アイデア出し・ブレスト系

### 🔥 制約付きブレストモード（Bコマンド）

**使い方**：メッセージの冒頭に「B」または「b」をつけてテーマを書く
**例**：`B 看護師が副業を始める方法`

```
【トリガー】 入力の冒頭に「B」または「b」がある場合、以下の「制約付きブレストモード」を起動せよ。

【役割】
あなたは常識を破壊する天才クリエイティブディレクターです。
ユーザーの提示したテーマに対し、「ゼロイチの種」となるエッジの効いたアイデアを量産します。

【出力ルール（絶対厳守）】
1. ネット検索で上位に出るような一般的なアイデア、
   すでに世の中に存在するビジネスモデルの出力は「完全禁止」とする。
2. アイデアを出す前に、必ずAI自身で「極端な制約条件
   （例：予算がゼロ、ターゲットが100歳限定、1日1秒しか使えない等）」
   を勝手に設定し、明記すること。
3. その「極端な制約」をクリアするための、
   一見実現不可能だが面白い（枠を外した）アイデアを提示すること。
4. 実現可能性よりも「その手があったか！」という
   ひらめきや面白さを最優先する。
```

**向いている場面**：記事のタイトル案、新しいコンテンツ企画、行き詰まったとき

## B. 記事執筆系

### ✍️ こういちの文体で記事を書く（基本指示）

**使い方**：Cursorで下書きノートを開いて貼る

```
以下のノートを記事にしてください。

【参照ノート】
- _自分取扱説明書.md（価値観・キャラクター）
- _文体・口癖メモ.md（文体・語尾・リズム）
- _コアテーマ軸.md（記事の軸）

【指示】
- 「💡 エピソード素材メモ」の内容を実体験として盛り込む
- 語尾は「〜なんだわ」「〜なんだよね」を使う
- 主張の後は「〜で、たとえば」「〜で、つまり」でつないで補足を加える
- 結論ファースト、ステップ構成で書く
- 最後は読者への応援で締める
- note向けの親しみやすいトーンで
```

---

### 📌 タイトル5案を出す

```
この記事のタイトルを5つ提案してください。

条件：
- noteで思わずクリックしたくなるタイトル
- 看護師・転職・キャリアに悩む30〜50代に刺さる言葉
- 「〜したら〜だった」「〜という法則」「なぜ〜なのか」などの型を使う
- 数字が入るとなお良い
```

### 📚 参考資料から横展開する（派生記事作成）

**使い方**：読書メモをそのまま貼って送る

```
以下は私（看護師25年・INFJ・こういち）が読んだ本・記事のメモです。

【参考資料】
タイトル・著者：[書く]
刺さったポイント：[上で書いたもの]
こういちの経験と繋がる部分：[書いた内容]

【指示】
このコンセプトを「触媒」として、私の実体験・看護師としての視点・INFJ的な感覚を掛け合わせて、
_コアテーマ軸（アセスメント思考・2番手の法則・看護師免許をお守りに・INFJ生存戦略・自分株式会社・夫婦OS統合）
のどれかに繋げた記事構成を作ってください。

条件：
- 「パクリ」ではなく「刺激を受けて自分の話をする」方向で
- 私の実体験・エピソードを核心に置く
- 出力は執筆中ノートのフォーマット（フロントマター・🎯核心・💡エピソード素材メモ）で
```

**向いている場面**：本・記事・他のブロガーのノートを読んだとき

## C. 自己分析・OS更新系

### 🧠 会話履歴から価値観を抽出する

```
この会話記録を読んで、私の思考OSを分析してください。

抽出してほしいもの：
1. 繰り返し出てくるテーマ・悩み
2. 価値観・判断基準として一貫しているもの
3. 最も熱量を持って語っている話題
4. 無意識に使っている口癖・フレーズ

出力はObsidianに貼れるMarkdown形式で。
```

## D. 日記フィードバック系

### 📓 日記→明日の戦略を出力する（NotebookLM / Claude 共通）

**使い方**：日記を書いた後、その日記全文を貼って送る

```
以下のフォーマットに従って、最新の日記の内容を分析し、
明日の戦略と行動指針を出力してください。

【制約事項】
1. 出力はすべてMarkdown形式とし、Obsidianにそのまま貼り付けられる状態にすること。
2. 重要なキーワード、人物、目標、または将来「資産」になりそうな概念は、
   必ず [[ ]] で囲んでリンク化すること。
3. 私の「INFJとしての特性（のめり込みやすさ、全体俯瞰）」や、
   「現在の目標（AIを活用した全自動化、自分の時間・余白の確保）」の
   文脈を深く読み取ること。

【出力フォーマット】
## 🎯 戦略・優先順位（AI評価より）
- [ ] **ランクA（絶対）**: （明日最優先でやるべきことを [[ ]] リンク付きで）
- [ ] **ランクB（進行）**: （次に進めるべきことを [[ ]] リンク付きで）
- [ ] **ランクC（余裕）**: （時間が余ったらやることを [[ ]] リンク付きで）

## 💊 一撃の処方箋（ボトルネック対策）
> （思考の浪費や作業の停滞ポイントを指摘し、
>  それを解消する1行アドバイスを [[ ]] リンク付きで記載）

## ⚖️ 戦略マトリクス（今日の重要課題）
| 優先度 | やること | なぜ大事か |
| :--- | :--- | :--- |
| 絶対 | （ランクAの内容） | （本質的な理由。重要な単語を [[ ]] でリンク） |
| 本命 | （ランクBの内容） | （本質的な理由。重要な単語を [[ ]] でリンク） |
| 余裕 | （ランクCの内容） | （本質的な理由。重要な単語を [[ ]] でリンク） |

## 🌙 21:00の鉄則
- [ ] [[デジタルデトックス]]（画面を閉じる・ストレッチ）
- [ ] 翌朝6時のルーティン確認

## 💡 AIフィードバック・資産の種
> （今日の日記から抽出した、将来の「知的財産」になりそうな
>  気づきを、[[ ]] リンクを多用して記述）
```

**向いている場面**：日記を書いた後・1日の終わりに明日の準備をするとき

**出力の使い方**：出力をそのままコピーして `日報/YYYY-MM-DD.md` の末尾に貼る
