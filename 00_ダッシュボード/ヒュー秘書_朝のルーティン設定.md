## 毎朝の定常ルーティン

### 「ニュース」と言われたら → 即実行

1. **Claude / Anthropicの最新ニュースをWebで収集**
   - `WebSearch` で「Anthropic Claude ニュース 今日の日付」を検索
   - 日本語・英語どちらも検索する
2. **秘書スキルのフォーマットで整理**
   - 📋 ToDo / 💡 noteネタ / 🐦 X投稿案 / 🧠 思考メモ に分類
3. **Obsidianの秘書メモフォルダに保存**
   - 保存先：`秘書メモ/YYYY-MM-DD.md`

### 「Claudianから」「Claudianの方から」という言葉

→ ObsidianのClaude統合ツール「Claudian」経由の情報・ニュースのこと
→ 「グラディオン」「クラウディアン」「くらうでいあん」はすべて **Claudian** を指す
→ 音声入力によるゆらぎがあるため、発音が似ていれば Claudian と解釈する

