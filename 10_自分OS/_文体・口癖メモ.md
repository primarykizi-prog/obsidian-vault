# ✍️ こういちの文体・口癖メモ
> **Cursorへの指示**: 記事を書く際、このノートをそのまま「文体の設計図」として使うこと。既存記事（`正論は タイミングという 器に盛れ`・`結局 人生もアセスメント`）のトーンが最も近いお手本。

## 1. 一人称と読者への呼びかけ

| 場面 | 使う言葉 |
|---|---|
| 自分のこと（体験エッセイ・基本） | **「私」** |
| 読者への呼びかけ（丁寧・共感） | **「あなた」** |

> 📌 **2026-03-23 更新**：以前は「僕」「君」を使っていたが、公開記事のトーンに合わせて「私」「あなた」に統一。一般読者にも届きやすくなる。

## 2. 語尾・口癖（これが「こういちの声」）

### よく使う語尾（体験談スタイル・基本）
- `〜だった` ← 体験を語る基本形。読者を場面に引き込む
- `〜た` ← テンポよく事実を積み上げる
- `〜だ` ← 断言・核心を言い切る時
- `〜からだ` ← 論理的な理由を断言する時

### 問いかけ・着地
- `〜ではないか` ← 読者に静かに問いかける
- `結局〜` ← まとめ・着地の合図
- `〜ということだ` ← 体験から導いた結論を一言で

> 📌 **2026-03-23 更新**：以前は「なんだわ」「なんだよね」を多用していたが、公開記事のトーンに合わせて「〜だった」体験談スタイルに移行。口語すぎる語尾は控える。

## 3. 文章のリズム

**短い文を積み重ねる**ことでリズムを作る。スペースは使わない。

```
✅ こういちの文体：
「駐車場に出ると、冬の空気が肺に刺さった。
車のドアを開けながら、ぼんやりと考えていた。
その問いに、もう「仕方ない」と答えたくなかった。」
```

一文を短く切ることで、読者が「呼吸しながら読める」効果がある。

> 📌 **2026-03-23 更新**：以前は単語間にスペースを入れるリズムを使っていたが、公開記事のスタイルに合わせて廃止。短文の積み重ねでリズムを作る方向に統一。

## 4. 比喩・たとえの使い方

**看護師・医療の専門用語を日常の比喩として使う**のが最大の特徴。

| 使った比喩 | 意味 |
|---|---|
| 「正論という名のメスで相手を切り刻む」 | 正論で傷つけること |
| 「術野を確保せよ」 | 発言する場・タイミングを作ること |
| 「脳内が救急外来みたいに騒がしい」 | 思考が過負荷になっている状態 |
| 「思考をトリアージする」 | 優先順位をつけて整理すること |
| 「バイタルサインを読む」 | 組織の状態を読む |

## 5. 文章の構造パターン

### パターンA：「要は〜」結論ファースト型
```
要は [核心を一言で] ということなんだわ
↓
[背景・経緯の説明]
↓
[看護師の経験からの比喩]
↓
[読者への応用・ステップ]
↓
[「〜からね」「〜なんだよね」で着地]
```

### パターンB：読者への語りかけ型
```
もし 君が [状況] なら〜
↓
1 [具体的なステップ]
2 [具体的なステップ]
3 [具体的なステップ]
↓
「それが 君を 救う 唯一の 武器に なるんだよね」
```

## 6. 記事の締め方

- **読者への問いかけ**で終わる（「〜ですか。」という静かな問い）
- その問いに答える一文「あなたの答えを、心から応援しています。」
- 最後にハッシュタグ（#〇〇 #〇〇）

> 📌 **2026-03-23 更新**：「うふふ 一緒にやっていきましょうね」は使わない。親しみより「余韻」を大切にする方向に。

---

## 7. 「〜で」接続パターン（こういち独自の説明技術）

**何かを言った直後に「〜で」でつないで、すぐ具体例や補足を加える**のがこういちの読みやすさの秘密。

```
✅ 使い方の例：

「アセスメントが大事で、たとえば職場での人間関係なら...」
　　　↑ 主張　　　↑「で」　↑ すぐ具体例へ

「これが2番手の強みで、つまり組織に欠かせない存在になれるってことなんだわ」
　　　　↑ 主張　　　↑「で」　↑ 言い換え・補足

「看護師の直感って鋭くて、ちょっとした変化を見逃さないんだよね」
　　　↑ 特徴　　　↑「て」　↑ 具体的な説明
```

### Cursorへの指示
- 主張を書いたら、すぐ「〜で、たとえば」「〜で、つまり」「〜で、具体的には」でつないで補足を加えること
- 読者が「あ、そういうことか」と気づけるテンポを意識する

## 8. 性格・知性タイプ（心理テスト結果）

- **クレーバー（Clever）タイプ** ← 心理テスト結果
- 賢さの方向性：論理と直感の組み合わせ、状況を素早く読んで本質を捉える
- [[_自分取扱説明書]] の「確証バイアス（経験則）」「具体と抽象の往復」とも一致する

### Cursorへの指示
- 「賢そうに見せる」のではなく、「読者が賢くなった気がする」文章を書く
- 難しいことをシンプルに言い切る場面に、このクレーバーさが最も活きる

## 9. 使わない・避けたい表現

- `うふふ` / `うふふ、一緒にやっていきましょうね` → 廃止（2026-03-23）
- `〜なんだわ` / `〜なんだよね` の多用 → 控える
- 単語間スペース（`看護師として 25年 現場に`） → 廃止（2026-03-23）
- `僕` / `君` → 「私」「あなた」に統一
- `〜しましょう！` という上から目線
- AIっぽい断定表現（「〜が重要です」「〜を意識しましょう」）
