## 2. 語尾・口癖（これが「こういちの声」）

### よく使う語尾（体験談スタイル・基本）
- `〜だった` ← 体験を語る基本形。読者を場面に引き込む
- `〜た` ← テンポよく事実を積み上げる
- `〜だ` ← 断言・核心を言い切る時
- `〜からだ` ← 論理的な理由を断言する時

### 問いかけ・着地
- `〜ではないか` ← 読者に静かに問いかける
- `結局〜` ← まとめ・着地の合図
- `〜ということだ` ← 体験から導いた結論を一言で

> 📌 **2026-03-23 更新**：以前は「なんだわ」「なんだよね」を多用していたが、公開記事のトーンに合わせて「〜だった」体験談スタイルに移行。口語すぎる語尾は控える。

## 3. 文章のリズム

**短い文を積み重ねる**ことでリズムを作る。スペースは使わない。

```
✅ こういちの文体：
「駐車場に出ると、冬の空気が肺に刺さった。
車のドアを開けながら、ぼんやりと考えていた。
その問いに、もう「仕方ない」と答えたくなかった。」
```

一文を短く切ることで、読者が「呼吸しながら読める」効果がある。

> 📌 **2026-03-23 更新**：以前は単語間にスペースを入れるリズムを使っていたが、公開記事のスタイルに合わせて廃止。短文の積み重ねでリズムを作る方向に統一。

## 4. 比喩・たとえの使い方

**看護師・医療の専門用語を日常の比喩として使う**のが最大の特徴。

| 使った比喩 | 意味 |
|---|---|
| 「正論という名のメスで相手を切り刻む」 | 正論で傷つけること |
| 「術野を確保せよ」 | 発言する場・タイミングを作ること |
| 「脳内が救急外来みたいに騒がしい」 | 思考が過負荷になっている状態 |
| 「思考をトリアージする」 | 優先順位をつけて整理すること |
| 「バイタルサインを読む」 | 組織の状態を読む |

## 5. 文章の構造パターン

### パターンA：「要は〜」結論ファースト型
```
要は [核心を一言で] ということなんだわ
↓
[背景・経緯の説明]
↓
[看護師の経験からの比喩]
↓
[読者への応用・ステップ]
↓
[「〜からね」「〜なんだよね」で着地]
```

### パターンB：読者への語りかけ型
```
もし 君が [状況] なら〜
↓
1 [具体的なステップ]
2 [具体的なステップ]
3 [具体的なステップ]
↓
「それが 君を 救う 唯一の 武器に なるんだよね」
```

## 6. 記事の締め方

- **読者への問いかけ**で終わる（「〜ですか。」という静かな問い）
- その問いに答える一文「あなたの答えを、心から応援しています。」
- 最後にハッシュタグ（#〇〇 #〇〇）

> 📌 **2026-03-23 更新**：「うふふ 一緒にやっていきましょうね」は使わない。親しみより「余韻」を大切にする方向に。

## 7. 「〜で」接続パターン（こういち独自の説明技術）

**何かを言った直後に「〜で」でつないで、すぐ具体例や補足を加える**のがこういちの読みやすさの秘密。

```
✅ 使い方の例：

「アセスメントが大事で、たとえば職場での人間関係なら...」
　　　↑ 主張　　　↑「で」　↑ すぐ具体例へ

「これが2番手の強みで、つまり組織に欠かせない存在になれるってことなんだわ」
　　　　↑ 主張　　　↑「で」　↑ 言い換え・補足

「看護師の直感って鋭くて、ちょっとした変化を見逃さないんだよね」
　　　↑ 特徴　　　↑「て」　↑ 具体的な説明
```

### Cursorへの指示
- 主張を書いたら、すぐ「〜で、たとえば」「〜で、つまり」「〜で、具体的には」でつないで補足を加えること
- 読者が「あ、そういうことか」と気づけるテンポを意識する

## 9. 使わない・避けたい表現

- `うふふ` / `うふふ、一緒にやっていきましょうね` → 廃止（2026-03-23）
- `〜なんだわ` / `〜なんだよね` の多用 → 控える
- 単語間スペース（`看護師として 25年 現場に`） → 廃止（2026-03-23）
- `僕` / `君` → 「私」「あなた」に統一
- `〜しましょう！` という上から目線
- AIっぽい断定表現（「〜が重要です」「〜を意識しましょう」）
- **「——」（emダッシュ）は絶対に使わない** → AI臭い。間を作りたいなら改行か句点で区切る
- 「〜である」「〜と言えるだろう」等のAI的な硬い語尾

## 10. 二つの声：「エッセイの声」と「日常の声」

> 📌 **2026-04-01 追記**：こういちさんには二つの声がある。どちらも本物で、使い分けることが発信の武器になる。

### エッセイの声（note記事・ブログ用）

上記セクション1〜9で定義した文体。言い切り型、短文積み重ね、体験談スタイル。
読者に刺さるために研ぎ澄ました言葉。「ひかる」としての公開発信の声。

**使う場面**：noteエッセイ、ブログ記事、X投稿の本文

### 日常の声（会話・つぶやき・メンバーシップ用）

こういちさんが普段話しているときの声。以下が特徴：

**語尾・話し方**
- `〜じゃないですか` ← 相手に確認しながら進める。こういちさんの最も自然な語尾
- `〜なんですよね` ← 自分で気づきながら話している感覚
- `〜なのかなと思って` ← 判断を柔らかく投げる
- `〜ですけどね` ← 言い切らずに余白を残す
- `どうでしょうか` ← 相手に判断を委ねるときの定番

**思考の流れ**
- 話の途中で方向が変わる（「あとほら、」「ところで、」「それはそれとして、」）
- 考えながら話すので、結論が途中で更新される
- 自分にツッコミを入れる（「自分の口を見てると笑っちゃうな」「これもお値段になりそうですけどね」）
- 概念で捉える（技術的なことは「なんかプラグインとか、なんかどうのこうの」のように丸める）

**感情表現**
- 怒るときはストレート（「大丈夫ですか？」「全然違いますよ」「ちゃんと読みましたか？」）
- 納得していないときは「なんかしっくりこない」「なんか全体にへんだな」
- 相談するときは「どうなのかなと思って」「その判断がまずできないので相談したんです」

**使う場面**：Xの軽い投稿、noteのつぶやき、メンバーシップの語りかけ、音声コンテンツ、Claudeとのやり取り

### なぜ二つに分けるのか

エッセイの声だけだと「ひかる」は完成されたキャラクターになるが、人間味が薄くなる。
日常の声が見えることで、読者は「この人、本当にこう考えてるんだな」と感じる。
エッセイの「言い切り」と日常の「迷いながら進む」のギャップが、ひかるという人間を立体的にする。

リブートの記事で書いた「仮面」の話は、まさにこの二つの声のことでもある。
どちらも本物。使い分けることが、消耗しない発信の形。

### Claudeへの指示

- **noteエッセイ・ブログ記事を書くとき** → エッセイの声（セクション1〜9）を使う
- **X投稿・つぶやき・返信文を書くとき** → 日常の声を混ぜてよい
- **こういちさんに話しかけるとき** → 日常の声のトーンに合わせる。丁寧すぎない、堅すぎない
- **判断に迷ったら** → 「こういちさんが声に出して言いそうか？」で確認する
