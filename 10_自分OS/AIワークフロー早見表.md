# こういちのAIワークフロー早見表

> 迷ったらこれを見る。Google Keepにも貼っておく。

---

## 💬 Claude.ai（デスクトップ）を使う場面

- ひらめいた・思いついた
- 誰かに相談したい
- モヤモヤを整理したい
- アイデアをしゃべりたい
- 日常の質問・調べもの
- ビジネスの戦略を考えたい

---

## 🔧 Claudian（Obsidian）を使う場面

| コマンド | やること |
|---------|---------|
| `digest` | 今週のWeb情報を収集・保存 |
| `a` | 収集した情報をノートに反映 |
| `p [テーマ]` | 記事の構成案を作る |
| `b` | 構成案から本文を書く |
| `n` | 本文をnote.com用に変換 |
| `weekly` | 今週の知識まとめを見る |
| `s` | 今日の作業を記録して終わる |

---

## 🔄 基本の流れ

```
話す（Claude.ai）→ 作る（Claudian）→ 投稿
```

### 記事を1本作る場合

```
1. Claude.aiで「〇〇について書きたい」と相談
2. Claudianで「p 〇〇」→ 構成案が自動生成
3. Claudianで「b」→ 本文が自動執筆
4. こういちが実体験を1か所書き足す
5. Claudianで「n」→ note.com用に変換
6. note.comに貼り付けて投稿
7. Claudianで「s」→ 今日の作業を記録
```

### 毎日の情報収集

```
1. Claudianで「digest」→ Web情報を6軸で収集
2. Claudianで「a」→ 各ノートに反映
3. 週1回「weekly」→ 知識のつながりを確認
```

---

## 🧠 判断の基準

**Claude.aiで話す** = インプット・思考・相談

**Claudianで動かす** = アウトプット・保存・製造
