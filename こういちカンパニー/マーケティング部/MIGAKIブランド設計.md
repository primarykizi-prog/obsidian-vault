## 🏯 体験設計｜「点から線」へ

> **2026-04-10 追記：大洲城泊事例からの知見**
>
> 愛媛県大洲市の城泊（1泊100万円超）は、訪日客数を2019年度比5.5倍にした。
> 成功の核は「ハードの豪華さ」ではなく、**「その場所でしか得られない物語的体験」**。
> また青森のスポーツツーリズムでは「点→線の体験設計」でリピーター増加に成功している。

### MIGAKIの「線」体験パッケージ構想

```
【糸魚川ストーリー体験パッケージ（仮）】

1点目：到着（山と海の交差点・MIGAKI BASEへ）
  → お父さんの「不動心」石碑との対話
  → 施設内の防具・竹刀・航海士の軌跡ディスプレイ

2点目：ジオパーク体験（ヒスイ海岸・フォッサマグナ）
  → 1500万年前から続く「磨かれた石（ヒスイ）」と「MIGAKI」を重ねる

3点目：食と地域（紅ズワイガニ・日本酒・日本海の夕日）
  → 地域と宿が「一体」になる体験

4点目：帰路（物語を持ち帰る）
  → ヒスイの小石、「不動心」の手書き書などのお土産体験
```

**設計原則**：「糸魚川に来た理由」を宿泊後に語れるようにする。
点（MIGAKI）ではなく線（糸魚川ストーリー全体）を体験させる。

## 💰 高単価化の根拠

> **2026-04-10 追記：城泊事例からの価格戦略根拠**
>
> 「高単価は非日常体験の質で正当化できる」という実証が城泊事例で確認された。
> 1泊100万円超でも欧米豪の富裕層に支持される理由は「唯一無二の物語」があるから。

- MIGAKIの唯一性：甲種一等航海士×剣道師範の父が作った場所→世界に1つ
- 競合が取れない差別化：「お父さんの講話体験（15分）」は他の施設が絶対に模倣できない
- 価格設定の方向性：「安さ」で勝負しない。「ここでしか得られないもの」で正当化する

## 🌏 インバウンド戦略

> **2026-04-10 追記：JNTO×糸魚川ワークショップの知見**
>
> JNTOが糸魚川を対象にインバウンド強化ワークショップを開催していることが確認された。
> 「国が後押しするエリアへの、競合が少ない今のタイミング」での参入は戦略的に正当性が高い。

### 糸魚川インバウンドの追い風

| 要素 | 詳細 |
|------|------|
| JNTO支援 | 糸魚川がインバウンド強化対象エリアとして国レベルで動いている |
| ジオパーク認知 | ヒスイ・フォッサマグナ・紅ズワイガニが差別化観光資源として登録済み |
| スキーシーズン | シーサイドバレー12/29〜3/22・シャルマン火打は5/5まで（長期対応可） |
| 競合の少なさ | 「武道×看護師×ジオパーク」の組み合わせは現在ほぼ空白 |

### 年間稼働モデル（構想）

```
【冬（12〜3月）】スキー×MIGAKI体験
  スキーゲスト＋リゾートワーケーション層

【春（4〜5月）】雪解けの静寂×新緑トレッキング
  ゴールデンウィーク＋リトリート層

【夏（6〜8月）】日本海×ジオパーク×グリーンシーズン
  欧米バイカー＋ジオツーリスト＋家族層

【秋（9〜11月）】紅葉×武道シーズン
  ツーリング×静寂ワーケーション層
```

> グリーンシーズン（特に夏・秋）の集客が年間稼働の安定鍵。
> インバウンドは夏〜秋を狙う。欧米富裕層は7〜9月が最大旅行シーズン。
