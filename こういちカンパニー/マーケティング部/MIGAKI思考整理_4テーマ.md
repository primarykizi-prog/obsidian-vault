### 体験の結晶化｜宿泊ストーリー設計

```
【到着】
外観は普通の民家。門をくぐると広大な日本庭園が現れる。
石碑に「不動心」。「あ、これは違う」が最初に起きる瞬間。

【夜】
グランドラウンジ（食堂）で地酒を開ける。
壁に世界地図と航海士のルート。棚に剣道の防具。
「この家の人たちは、どんな人生を生きてきたんだろう」と思い始める。

【朝】
庭に出ると、お父さんが静かに鯉に餌をやっている。
ゲストの前には出ない。でも庭の向こうに気配がある。
その光景が、何かを思い出させる。

【チェックアウト後】
枕元に手紙がある。英語で書かれている。
「Fudoshin. May you carry it with you.」
```

## テーマ② 経営・戦略
### 看護師オーナーの挑戦：無借金・個人事業で1泊10万円を勝ち取る「勝機」の設計図

### 1泊10万円の根拠

| 条件 | 内容 |
|---|---|
| ターゲット | 欧米・オセアニアの富裕層スキーヤー（9泊以上・グループ） |
| 客室 | 50人収容・2棟（グループ貸切） |
| 競合 | 白馬エリアは混雑・高騰。「白馬の隣の静かな別邸」として差別化 |
| 訴求 | 庭園・不動心・Luxury Dojo体験・看護師ホストの安心 |
| 週単位 | 1泊8万円 × 7泊 → 20〜25%割引 → 約42〜45万円/週 |

### 無借金が生む「攻め」の経営

借金がないから、安くする必要がない。安くしないから、高単価ターゲットだけを狙える。
高単価ターゲットだけを狙うから、クオリティに全力を集中できる。
これが好循環の設計図だ。

### 補助金戦略（最大活用後の実質負担）

| 項目 | 補助金前 | 補助金後（想定） |
|---|---|---|
| 断熱・内窓 | 100〜150万円 | 実質30〜50万円 |
| キッチン全面リノベ | 50〜100万円 | 実質10〜30万円 |
| 乾燥室設置 | 20〜40万円 | 実質5〜15万円 |
| 水回り | 80〜120万円 | 実質20〜40万円 |
| **合計** | **250〜410万円** | **65〜135万円** |

### ポジショニングの正当性

「白馬の喧騒を避ける隠れ家」は、白馬のオーバーツーリズム問題（混雑・価格高騰・外国人からも批判あり）が追い風になっている。
糸魚川は白馬から車1時間。「近い・静か・本物の日本体験」の三点で差別化できる。
