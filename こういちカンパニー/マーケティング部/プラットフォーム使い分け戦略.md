# プラットフォーム使い分け戦略｜こういちカンパニー
作成日：2026-03-29
関連：[[こういちカンパニー/ダッシュボード]] / [[差別化戦略_ライダーズベース構想]]

---

## 🎯 大原則

> **noteで認知、リベで窓口。自分でコミュニティを作らず、質が担保された場所に乗る。**

---

## 🔑 核心：「認知」と「窓口」の分離

```
【認知（入り口）】         【窓口（質の担保）】
  note ──────────────→ リベシティ
  X                       ↑
  Airbnb                  │
                          │ 月額2,000円が
                          │ 自然なフィルターになる
                          │
                     本気の人だけが残る
```

**note** = 「知ってもらう場所」
- 誰でも無料で読める
- 「看護師が糸魚川で民泊やってるらしい」と知ってもらう
- ストーリー・体験談・活動報告

**リベシティ** = 「本気の人が来る窓口」
- note記事の最後に「詳しくはリベシティで」と誘導
- 「自分もやりたい」「相談したい」人はリベに来てもらう
- リベの月額が質のフィルター → こういちが審査しなくていい
- 経験者からのフィードバックも同時に得られる

**この設計のメリット：**
- 自分で有料コミュニティを作る必要がない（運営コスト・集客コストがゼロ）
- リベシティの信用がそのまま自分の信用になる
- 「リベの民泊やってる人」というポジションが取れる
- 住職ルートと同じ「広告費ゼロで質の高い人が集まる」構造

## プラットフォーム別の役割

---

### 📝 note（認知・入り口）

**役割：** 「この人面白い」「こういう民泊があるんだ」と知ってもらう場所

| 項目 | 内容 |
|---|---|
| **なぜnoteか** | すでにひかるカンパニーで運用中 → ノウハウがある。noteは「体験の共有」が読まれやすい |
| **使い方** | 民泊の開業過程をシリーズ記事として発信 → 看護師・移住希望者・ライダーの目に留まる |
| **頻度** | 月1〜2本（大きな進捗があった時に書く） |
| **読者** | 看護師コミュニティ・地方移住に興味がある人・副業を考えている看護師 |

**noteに出すもの：**
- 開業のリアルストーリー（「住職に挨拶に行った日」「管理会社を選んだ理由」等）
- 看護師が副業で民泊をやるメリット・デメリット
- 糸魚川の魅力紹介（写真付き）
- 開業後の月次振り返り（数字は出しすぎない）

**noteの記事の最後に必ず入れる一文：**
> 「民泊のリアルな相談・情報交換はリベシティでやっています。興味がある方はぜひ。」

**noteに出さないもの：**
- 具体的な収支の数字（→ リベシティで）
- 専門的な相談（→ リベシティで）
- 集客のDM的な告知（→ Airbnb・SNSで）

---

### 🏠 リベシティ（窓口・質の担保）

**役割：** 本気で民泊・地方移住をやりたい人が来る場所。相談も受けるし、相談もする。

| 項目 | 内容 |
|---|---|
| **なぜリベか** | 月額2,000円の有料コミュニティ → 本気の人しかいない → 質のフィルターが自動でかかる |
| **2つの使い方** | ①自分が相談する（壁打ち・フィードバック） ②興味を持った人の受け皿になる |
| **投稿先** | 不動産チャット / 事業チャット / 副業チャット |

**リベシティに出すもの：**
- 物件ワンペーパー（相談用）→ [[物件ワンペーパー_相談用]]
- 管理会社の選定相談
- 旅館業許可の進捗共有
- 収支シミュレーションの妥当性確認
- 看護師×民泊モデルへの意見募集
- noteから来た人への詳しい情報共有

**リベシティの窓口としての使い方：**
- noteで認知した人が「もっと聞きたい」と思ったらリベに来てもらう
- リベ内で「糸魚川の民泊の人」というポジションを確立する
- 将来的にはリベ内で「民泊オーナー仲間」のネットワークができる

---

### 🐦 X（拡散・認知）

**役割：** noteやリベの活動を拡散する＋ライダーコミュニティとの接点

| 項目 | 内容 |
|---|---|
| **使い方** | note記事のシェア / 開業の進捗つぶやき / ライダー向け情報発信 |
| **頻度** | 週2〜3回（軽いつぶやき） |
| **ハッシュタグ** | #看護師民泊 #糸魚川 #ライダーズベース #シーサイドバレー #地方移住 #看護師副業 |

---

### 🏍️ Airbnb・予約サイト（直接集客）

**役割：** 実際にゲストが予約する場所

| 項目 | 内容 |
|---|---|
| **使い方** | 物件ページの最適化・レビュー対応 |
| **差別化ポイント** | 「看護師夫婦が運営」「風巻神社推薦」をプロフィールに明記 |

---

## 🔄 プラットフォーム間の連携フロー

```
【こういちの行動・体験】
  │
  ├─→ リベシティ（判断前の相談・壁打ち）
  │     └→ フィードバックを受けて意思決定
  │
  ├─→ note（行動した結果をストーリーとして発信）
  │     └→ 読者が増える → 看護師・移住希望者が認知
  │
  ├─→ X（noteの拡散 ＋ ライダー向け情報）
  │     └→ 認知が広がる → 予約につながる
  │
  └─→ Airbnb等（実際の予約）
        └→ 売上 → 月次レポート → 次の改善
```

**ポイント：「noteで知ってもらう → Xで広げる → リベで深くつながる → 予約サイトで売上」という流れ。認知と窓口を分けることで、こういちの負担がゼロに近い。**

## ⚠️ やってはいけないこと

- ❌ 自前で有料コミュニティを作る（質の担保が難しい・運営コストが高い）
- ❌ noteで収支の詳細を全公開する（リベシティでやるべき内容）
- ❌ リベシティでストーリー的な日記を書く（場違い・noteでやるべき）
- ❌ 全プラットフォームに同じ内容を投稿する（各プラットフォームの役割が違う）
