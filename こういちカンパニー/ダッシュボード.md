## 組織図

```
こういち（CEO）
  │ 指令・判断・意思決定のみ
  │
  ▼
秘書（Claude）── 唯一の窓口・振り分け・組織管理
  │
  ├── 予約管理部
  │   └ カレンダー管理・空き確認・ダブルブッキング防止
  │
  ├── ゲスト対応部
  │   └ ウェルカムメッセージ・FAQ・レビュー返信下書き
  │
  ├── 財務管理部
  │   └ 収支記録・月次レポート・収益シミュレーション・補助金管理
  │
  ├── マーケティング部
  │   └ チラシ更新・SNS投稿案・Airbnbページ最適化・季節打ち出し
  │
  └── 現地連携部
      └ 両親・管理会社・住職・病院との連絡調整
```

**人間チーム（AI社員と連携）：**
```
管理会社（運営委託先）
  └ 予約受付・清掃・ゲスト現地対応

お父さん・お母さん（現地責任者）
  └ チェックイン対応・施設見回り・緊急一次対応

住職・風巻神社（集客パートナー）
  └ ライダーへのチラシ配布・口コミ・イベント情報提供

糸魚川総合病院（採用連携パートナー）
  └ 看護師試住プログラムの紹介先

奥さん（家庭内CFO・相談パートナー）
  └ 大きな意思決定の壁打ち・家計への影響確認
```

## 情報の流れ

```
ゲストの問い合わせ・予約
  ↓
管理会社 → 秘書（Claude）に報告
  ├→ 定型対応 → ゲスト対応部が自動対応
  └→ 判断が必要 → こういちに確認依頼
```

```
こういちの指令・気づき・思いつき
  ↓
秘書（整理・分類・振り分け）
  ├→ 予約関連 → 予約管理部
  ├→ ゲスト関連 → ゲスト対応部
  ├→ お金関連 → 財務管理部
  ├→ 集客関連 → マーケティング部
  └→ 現地関連 → 現地連携部
```

```
月次レビュー（月1〜2時間）
  ↓
財務管理部 → 月次レポート提出
マーケティング部 → 稼働率・集客レポート提出
  ↓
こういちが判断・次の方針を秘書に伝える
```
