## 二段構えのファン設計

```
【無料ゾーン｜ひかるの世界】
  note無料記事：看護師の消耗・INFJ・転職・自己磨き
  X投稿：日常・気づき・MIGAKI進捗（少し）
  Instagram Stories：物件改装・庭・リアルタイム

  ↓ 記事末尾に「橋渡し一文」を挿入

【有料ゾーン｜MIGAKI道場】
  開業の全記録・失敗談・収益公開・哲学
  白帯（500円）→ 青帯（980円）→ 黒帯（3,000円）
```

## メンバーシップ名：「MIGAKI道場」

### 会員の呼び名：「道友（どうゆう）」

同じ道を歩む友。読者ではない、仲間だ。

### 開設メッセージ（note掲載用）

```
MIGAKI道場へようこそ。

ここは、消耗した人が来る場所ではありません。
「磨こう」と決めた人が来る場所です。

私の父は、世界の海を渡り、最後に竹刀を握った。
庭に「不動心」と刻んだ石を置いた。

私はその石の前で決めました。
「場所を磨く」と。

あなたと一緒に、磨いていきます。
稽古を、始めましょう。

--- こういち（MIGAKI CEO）
```

## 3段階プラン設計

### 🥋 白帯（Shiro）｜月額500円

**コンセプト**：「共感から参加へ」

| 特典 | 内容 |
|---|---|
| MIGAKI開業日誌（全話） | 保健所との戦い・庭の鯉・家族の決断（リアルな記録） |
| 月次レポート | 進捗・数字・今月の気づき（1,000〜2,000字） |
| 先行公開 | 無料記事の2週間前に読める |

**こんな人に**：「ひかるの記事が好きで、もう少し深く知りたい人」

---

### 🥋 青帯（Ao）｜月額980円

**コンセプト**：「経営の裏側まで共有する」

| 特典 | 内容（白帯に加えて） |
|---|---|
| ボツ案・失敗記録 | 「やらなかったこと」「間違えたこと」の完全公開 |
| 収支の透明化 | 開業費・補助金額・月次売上（段階的に公開） |
| CEO質問コーナー | 月1回、道友からの質問にこういちが直接回答 |
| 施工中の写真・動画 | DIYの現場・ビフォーアフターのリアル |

**こんな人に**：「副業・民泊・独立を真剣に考えている人」「経営の現実を知りたい人」

---

### 🥋 黒帯（Kuro）｜月額3,000円

**コンセプト**：「道友はMIGAKIの一部になる」

| 特典 | 内容（青帯に加えて） |
|---|---|
| **MIGAKI BASE 優先予約権** | 道友は一般予約の1ヶ月前から予約できる |
| **「不動心合宿」招待** | 年1回・MIGAKI BASEに泊まる特別体験（道友限定） |
| **オーナーメッセージ** | こういちから直接のDM・近況報告 |
| **開業記念プレート** | 「初期道友」として名前を施設に刻む（検討中） |

**こんな人に**：「MIGAKIを本気で応援したい人」「実際に泊まりたい人」

> 💡 **黒帯の道友は、MIGAKIに泊まれる。** これが最強の特典。
