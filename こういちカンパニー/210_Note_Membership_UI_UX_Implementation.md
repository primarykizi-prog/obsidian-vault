## 🔑 【緊急追加】鉄筋の塔——ブランドの核心資産

> **日曜リリース前に、全員が把握すべき最重要情報。**

### 父の二重の専門性

| 資格 | 意味 |
|------|------|
| **甲種一等航海士** | 日本の船員免許の最高位。国際航海の実務経験。英語対応可能 |

これは「元民宿のオーナー」ではない。
**「世界の海を知り、建物の構造を知り、不動心の石を庭に置いた人間」**がホストである。

### 鉄筋の塔——MIGAKI BASEのシンボル

敷地内の離れに、一棟の塔がある。

鉄筋構造。一人で作った。
設計図も、重機も、職人も使わずに——父が、独力で。

それがどれほどの仕事か、建築を知る人間ならわかる。
一級土木建築士の資格を持つ人間が、自分の手だけで鉄筋を組み上げ、コンクリートを流し込んだ。
結果として、唯一無二の「手作りの要塞」が糸魚川の山の中に存在する。

**この塔は「資産」ではなく、「物語」だ。**

## 🎨 Mission 1｜メンバーシップの視覚的ブランド設計

### 1-1 配色方針——「Nothing is Luxury」のビジュアル化

**テーマ：Quiet Darkness（静かな暗さ）**

noteのメンバーシップ設定に直接入力できる配色基準：

| 要素 | 指定値 | 理由 |
|------|--------|------|
| **ベース** | 黒（#0D0D0D）または深い墨色（#1A1A1A） | 余計な情報を消す。視線がテキストだけに集まる |
| **アクセント** | ディープネイビー（#1B3A5C）または鉄紺（#1F3045） | 日本海の深さ・甲種航海士の制服・夜の外洋 |
| **テキスト** | オフホワイト（#F2F2F0） | 白すぎない。読み続けられる温度 |
| **禁止色** | 明るいオレンジ・ピンク・過剰なグリーン | MIGAKIブランド禁止事項（「かわいい」方向に引っ張る色） |

**note設定での実装：**
- カバー画像の背景色はダーク系に統一
- サムネイル画像は必ず「加工なし・フィルターなし」
- フォントカラーは自動（noteのデフォルト）で問題なし——記事内容で勝負

### 1-2 ヘッダー画像——選定基準と具体的候補

**原則：「磨かれた静寂」を写す。人物を写さない。**

| 優先順位 | 被写体 | 撮影タイミング | 意図 |
|---------|--------|--------------|------|
| 🥇 **最優先** | **鉄筋の塔の外観** | 早朝・曇天・霧がかかった状態 | 唯一無二性・父の狂気的情熱を視覚化 |
| 🥈 | 「不動心」の石碑 | 夜明け直後・逆光 | 哲学の核心を一枚で表現 |
| 🥉 | 根知地区の一本道 | 霧・冬の静寂 | 「何もない」という贅沢の表現 |
| 参考 | ガレージの工具 | 手前だけにピント・背景ボケ | 職人性・手仕事の温度 |
| 参考 | 雪に埋もれた庭 | 深夜〜早朝 | 静寂ツーリズム・Zero Frictionの視覚化 |

**ヘッダー画像のサイズ指定（note仕様）：**
```
推奨サイズ：1280 × 670px（16:9）
最低サイズ：960 × 504px
形式：JPG（容量5MB以下を推奨）
圧縮：TinyPNGなどで軽量化してからアップロード
```

**画像がない場合の代替手段：**
1. Canvaで「黒背景 + 白細字"MIGAKI BASE"」のミニマルヘッダーを作成
2. 文字のみ構成で十分——MIGAKIは「写真で売る」ブランドではなく「言葉で売る」ブランド

### 1-3 プランアイコン設計——SF Symbols風ラインアートの統一

**統一原則：線の太さ1〜2px・余白多め・装飾なし**

| プラン | アイコン案 | 代替絵文字 | 意味の説明 |
|-------|-----------|-----------|-----------|
| 🧭 **航海士（¥500）** | コンパス（円形・針のみ） | 🧭 | 「方向を見つける旅人」——観察者であり、仲間 |
| 🏠 **住人（¥120,000〜）** | ランタン（炎なし・枠だけ） | 🕯 | 「暗い中で灯りを持つ人」——この場所を選んだ人 |
| ⚓ **提督（個別）** | 舵輪（ラット・6本スポーク） | ⚓ | 「航路を決める人」——事業を共に設計する者 |

**なぜSF Symbols風ラインアートで統一するか：**

IT業界のツール（Linear、Notion、GitHub）が採用するこのスタイルは
「洗練・機能性・余計なものを持たない」の象徴だ。
MIGAKIのターゲットであるITワーカーは無意識にこのデザイン言語を「信頼」と結びつける。
逆に言えば、カラフルなアイコンや3Dアイコンは「安っぽさ」のシグナルになる。

**アイコン素材の入手方法（無料）：**
- Noun Project（https://thenounproject.com）で「compass line」「lantern outline」「ship wheel」検索
- ダウンロードしたSVGをCanvaに貼り付け→白色変換→黒背景に配置
- 出力：300×300px PNG（透過あり）

## 🔄 Mission 2｜運営フローの構造化

### 2-1 コンテンツの二層構造

```
【ストック層：限定マガジン】永久に価値が残るもの
       ↕（毎月の積み上げ）
【フロー層：掲示板】鮮度が命・今日の格闘
```

### 2-2 限定マガジン（ストック）——格納ルール

**名称：「MIGAKI航海日誌」**

| 投稿頻度 | 月2本（最低ライン）〜月4本（理想ライン） |
|---------|----------------------------------------|
| 文字数 | 1,500〜3,000字（読み切れる長さ） |
| 公開対象 | 航海士プラン（¥500）以上の全会員 |
| 投稿日 | 毎月10日前後・25日前後（給与日・月末を避ける） |

**マガジン記事のジャンル定義：**

| ジャンル | 内容 | 頻度 |
|---------|------|------|
| 📋 行政格闘記 | 保健所・消防・補助金の実況。様式を何回直したか。担当者が何を言ったか | 月1本 |
| 🔨 施工日誌 | じゅん君との格闘・父の一言・鉄筋の塔の秘密 | 月1本 |
| 💭 哲学ノート | 看護師として見てきた「疲弊」とMIGAKIの関係。CEOの思想の深部 | 月1〜2本 |
| 🏥 リカバリー知見 | 救急26年の経験から引き出す「回復の設計論」——住人プランの土台 | 随時 |

**鉄筋の塔コンテンツ設計（目玉連載）：**
```
第1話：「父が塔を建てた理由を、私は知らなかった」
第2話：「一級土木建築士が、なぜ一人で鉄筋を組んだのか」
第3話：「あの塔の中で、あなたは何を書くか」（住人プランへの接続）
```

---

### 2-3 掲示板（フロー）——「今日の格闘」

**原則：写真1枚＋短文200字以内。完結させなくていい。**

| 投稿頻度 | 週2〜3回（月曜・水曜・金曜が基本） |
|---------|----------------------------------|
| 投稿時間 | 夜22:00〜23:00（看護師シフト終了後） |
| 公開対象 | 航海士プラン以上の全会員 |

**掲示板の投稿テンプレート（コピペして使う）：**

```
【今日の格闘｜◯月◯日】

（写真：工事・書類・庭・父の一言 等）

今日の出来事を1〜3文で書く。
完結させなくていい。続きは次回。

── こういち
```

**投稿の例文（雰囲気参照用）：**
```
【今日の格闘｜4月10日】

保健所から返事が来た。
今度は「換気の計算書が足りない」と。

26年間、2mmの誤差が命を変える現場にいた私が
今は2cmの壁の厚さを直している。

なぜか、笑えてきた。

── こういち
```

### 2-4 免責事項——心理的安全性を守る文案

**場所：メンバーシップのトップページ説明文の末尾に固定掲載**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
ご参加にあたっての正直な告白
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

私はいま、MIGAKI BASEを一人で作っています。
看護師として働きながら、行政と格闘しながら。

コメントにはできる限りお返事しますが、
現場作業が続く期間は「スキ」のみになることがあります。

それでも——毎週、記事を書きます。
現場の正直な景色を、お届けし続けます。

月500円という小さな関わりが、
MIGAKI BASEを前進させる燃料になります。

── こういち（MIGAKI CEO・看護師26年目）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**この文章が果たす3つの機能：**
1. CEOが「返せない時期」への心理的プレッシャーを事前に解消する
2. 読者に「この人は正直だ」という信頼を生む（逆説的に親近感が増す）
3. 「現場優先」という姿勢がMIGAKIブランドの真摯さを強化する

## 💬 Mission 4｜入会誘導文テンプレート（3パターン）

---

### パターン A｜静かな招待型（最も汎用性が高い）

```
---

少しだけ、話してもいいですか。

私はいま、新潟県糸魚川市の山の中で宿を作っています。
一級土木建築士の父が一人で建てた鉄筋の塔がある、その場所で。

毎週、保健所との格闘を書いています。
補助金の申請書を何回書き直したかも、全部書きます。

「航海士」として、見ていてもらえたら嬉しいです。
月500円。初月は無料です。

→ MIGAKI道場（航海士プラン）へ

---
```

---

### パターン B｜物語接続型（記事の感情の流れを活かす）

```
---

今日の話は、ここまでです。

続きは——宿の工事が始まってから、書けるようになったことがあります。
看護師として「消耗」を見てきた私が、今「回復の設計」をしている現場のことです。

毎週、MIGAKI航海日誌に書いています。

月500円。初月無料。
父が独力で建てた「鉄筋の塔」の正体も、そこで話します。

→ MIGAKI道場（航海士プラン）へ

---
```

---

### パターン C｜ファクト直球型（ITワーカー・起業検討者向け）

```
---

【航海日誌（会員限定）で公開していること】

・保健所に4回落とされた時の書き直し内容
・補助金200万円の申請書を看護師が書いた全記録
・一級土木建築士の父が一人で鉄筋構造の塔を建てた理由

「本気で関心がある人だけ」が集まる場所を作っています。

→ MIGAKI道場（航海士プラン）へ

---
```

---

**3パターンの使い分け：**

| パターン | 使う記事 | 狙うターゲット |
|---------|---------|--------------|
| A（静かな招待） | INFJ・働き方・看護師消耗系の記事 | 共感ベースで動く読者 |
| B（物語接続） | 個人的体験・エッセイ系の記事 | 読後の感情が高い読者 |
