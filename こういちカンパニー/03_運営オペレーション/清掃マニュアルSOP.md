## 基本方針

> **「楽しみ」を「苦労」に変えない仕組みを先に作る。**
> 口頭での引き継ぎは崩壊する。チェックリストで体制を維持する。

**担当体制**：
- **一次清掃**：ご両親（ゲストチェックアウト後・確認対応）
- **最終仕上げ**：こういち夫婦（週1〜2回訪問時・品質検品）
- **繁忙期の外注**：清掃業者（1回15,000〜30,000円・繁忙期のみ）

---

## 🔴 チェックアウト後の清掃フロー（通常）

### STEP 1：ご両親による一次清掃（60〜90分）

**寝室・和室**
- [ ] 布団・シーツ・枕カバーを外す（洗濯へ）
- [ ] 畳の目に沿って掃除機をかける
- [ ] 窓を開けて換気（15分以上）
- [ ] 備品（アメニティ等）の数量確認・補充
- [ ] ゴミ箱のゴミを回収

**バスルーム・洗面台**
- [ ] バスタブ・シャワー・床の洗浄
- [ ] 鏡・洗面台を拭き上げ（水垢なし）
- [ ] タオル類を回収（洗濯へ）
- [ ] トイレ便座・便器・床の清掃
- [ ] アメニティ類の補充確認

**キッチン・食堂**
- [ ] 調理台・シンクの洗浄・乾拭き
- [ ] 使用食器の洗浄・所定位置への戻し
- [ ] 床の清掃
- [ ] Honesty Barの在庫確認・補充リスト作成

**共用部・玄関**
- [ ] 廊下・玄関の掃き掃除・モップ
- [ ] スリッパを整頓・消毒
- [ ] 靴箱エリアの確認

**庭**
- [ ] 庭の見回り（落ち葉・ゴミの確認）
- [ ] 鯉の池の状態確認
- [ ] 洗濯物・私物が出ていないか確認（NG）

### STEP 2：こういち夫婦による最終仕上げ（30〜45分）

**品質検品（看護師目線）**
- [ ] 畳の目・フローリングの拭き残し確認
- [ ] 窓ガラス・鏡の指紋・水垢確認
- [ ] 布団のセッティング（縫い目の向き統一）
- [ ] アメニティの配置（位置・向きを統一）
- [ ] 不動心の石碑周辺の最終確認
- [ ] 写真撮影（Airbnb品質管理用）

**感染症・衛生管理（看護師監修プロトコル）**
- [ ] ドアノブ・電灯スイッチ・リモコンのアルコール消毒
- [ ] 共用トイレの便座・フタのアルコール消毒
- [ ] バスルームの排水溝確認
- [ ] キッチン調理台の食品残渣確認

## 🚨 緊急対応フロー（設備故障時）

```
ご両親が異常発見
  │
  ├→ 写真撮影してLINEでこういちに送信
  │
  こういちが判断
  ├→ 軽微（電球切れ等）→ ご両親に対応依頼
  └→ 重大（水漏れ・設備故障等）→ 業者手配
        │
        ゲストへのメッセージ送信（AIが対応）
        └→ 「問題が解決されました。ご不便をおかけしました。」
```
