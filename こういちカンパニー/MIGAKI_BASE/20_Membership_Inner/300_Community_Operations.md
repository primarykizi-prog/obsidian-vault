# MIGAKI BASE コミュニティ運営設計書
## 「チームMIGAKIの掟」

---

## 1. コミュニケーションルール（CEOの約束と限界）

### CEOからの返答ルール

```
【チームMIGAKIへ】

私からの返答は、週1回・まとめて行います。

毎週月曜日か火曜日に、
前週の掲示板の投稿・コメント・質問に対して
まとめて返答します。

「すぐに返事がほしい」という場所ではありません。
ゆっくり考えて、ゆっくり返す。
それがこの場所のペースです。

緊急の相談は、掲示板のタイトルに【急ぎ】と
つけてください。できるだけ早く確認します。
```

### 相互扶助の指針（チームMIGAKIの行動規範）

```
チームMIGAKIには、3つのお願いがあります。

① 知っていたら、教える
  行政対応・旅館業法・補助金・建築・DIY——
  もし経験があれば、遠慮なく投稿してください。
  「素人の意見ですが」でも十分です。
  その一言が、このプロジェクトを動かすことがあります。

② 「いいね」だけでも価値がある
  何も書けなくても、反応するだけで伝わります。
  「見ている人がいる」という事実が、
  CEOを前に進ませます。

③ 答えを求めない
  ここは「解決する場所」ではなく「考える場所」です。
  誰かの投稿に対して「それは違います」ではなく、
  「私はこう感じました」という形で話してください。
```

## 2. 掲示板 投稿テンプレート

### 【CEO用】月初「今月の相談」テンプレート

```
タイトル：【今月の相談】〇月：〇〇について教えてください

━━━━━━━━━━━━━━━━
今月、詰まっていること
━━━━━━━━━━━━━━━━

〔今月の具体的な課題を1〜2文で〕

例：
「保健所から『この書類が足りない』と言われました。
 ○○という書類の書き方がわかりません。」

━━━━━━━━━━━━━━━━
もし知っていたら教えてほしいこと
━━━━━━━━━━━━━━━━

〔具体的な質問を1つだけ〕

例：
「旅館業の許可申請で、同じ経験をされた方はいますか？
 または、行政書士への依頼を経験された方がいれば
 費用感だけでも教えていただけると助かります。」

━━━━━━━━━━━━━━━━
今月の進捗（報告）
━━━━━━━━━━━━━━━━

〔先月からの変化を1〜3文で〕

知恵を貸してくれた方、ありがとうございました。
```

### 【CEO用】週次「格闘日誌」テンプレート

```
タイトル：【格闘日誌 #〇〇】〇月〇週：〇〇と向き合った話

今週、一番時間を使ったこと：
→ 〔1文で〕

今週、一番驚いたこと：
→ 〔1文で〕

来週、やること：
→ 〔1文で〕

今週の光（小さくてもいい）：
→ 〔1文で〕
```

---

### 【メンバー用】はじめての投稿テンプレート

```
タイトル：はじめまして。〔名前 or ニックネーム〕です。

はじめまして。
〔名前 or ニックネーム〕と申します。

このメンバーシップに入ったきっかけ：
→ 〔自由に〕

私が「光を失いかけた」と感じた瞬間（もしよければ）：
→ 〔自由に、書かなくてもOK〕

MIGAKI BASEに期待していること：
→ 〔自由に〕

よろしくお願いします。
```

---

### 【メンバー用】知恵・情報提供テンプレート

```
タイトル：【情報提供】〔テーマ〕について知っていること

CEOの〔○月の相談 or ○○という投稿〕を読んで、
少し知っていることがあったので書きます。

〔知っていること・経験・感じたことを自由に〕

専門家ではないので参考程度に。
でも、役立てばうれしいです。
```

## 3. ウェルカムメッセージ（入会直後に表示）

```
━━━━━━━━━━━━━━━━
チームMIGAKIへ、ようこそ。
━━━━━━━━━━━━━━━━

入ってくださって、ありがとうございます。

まず、正直に言います。

この場所は、完成していません。
建物は改装中で、許可もまだ下りていません。
行政との格闘は、今もリアルタイムで続いています。

それでも入ってくださったということは——
あなたは「完成品を買いに来た客」ではなく、
「作っている過程を一緒に面白がる人間」だということです。

だから、お願いがあります。

━━━━━━━━━━━━━━━━
あなたに期待していること
━━━━━━━━━━━━━━━━

① 見ていてください
  格闘の記録を読んで、「そうか、大変なんだな」と
  思ってもらえるだけで、私は前に進めます。

② もし知っていたら、教えてください
  旅館業法・建築・行政・資金調達——
  何でも構いません。素人の経験談でも十分です。

③ 「使いたい」と思ったら、言ってください
  完成したとき、あなたに真っ先に声をかけます。

━━━━━━━━━━━━━━━━
私が約束すること
━━━━━━━━━━━━━━━━

① 嘘をつきません
  うまくいかないことも、そのまま書きます。

② 毎週、更新します
  どんなに小さな進捗でも、記録し続けます。

③ あなたの意見を、この場所に反映します
  毎月1問アンケートの答えが、設計に直接使われます。

④ いつか、本物の鍵を渡します
  最初の仲間に、この建物の実際のドアキーを。
  それが、私からの最大の感謝の形です。

━━━━━━━━━━━━━━━━
最初の投稿について
━━━━━━━━━━━━━━━━

掲示板に「はじめましての投稿テンプレート」があります。
気が向いたら、ひとこと書いてみてください。

書かなくても、ここにいてくれるだけでいいです。

── こういち（MIGAKI BASE CEO）

MIGAKI BASE — 幸せを、磨く。
```
