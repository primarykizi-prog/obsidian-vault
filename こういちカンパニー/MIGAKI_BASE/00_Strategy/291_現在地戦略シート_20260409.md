## 🎯 日曜ミッション：要点3つ

### ① 入口の設計（v5確定構造）
近況報告（noteフォロワーへの感謝）
→ 治る/元に戻るの宣言
→ 場所の物語（父の遺産）
→ 格闘中の誠実な告白
→ メンバーシップ案内
→ 【メンバーと一緒に作っていきたいこと（予定）】（こういち原文・書き換え禁止）
→ 鍵の約束（第0期メンバーへ）
→ クロージング

### ② クロージング直前の一押し（追加推奨）
> 「日曜の夜、翌日を考えて少し重くなる感覚がある人に、届けたい。」

### ③ 掲示板を先に埋める（最重要）
ボタンを押す前に「中がある」と見せること。
開設当日・掲示板第0投稿を先に書く。
例：「今日、保健所から想定外の回答が来ました。詰まっています。」
これが転換率を最も左右する。

## 🎨 ビジュアル戦略：アンチグラビリティ×なのバナナ2

### 判定：条件付きYES——使う場所を間違えるな。

**二層構造の設計**

| 用途 | トーン | ツール |
|-----|--------|--------|
| Phase 2 ライダー集客広告・SNS・ポスター | アンチグラビリティ（一本道・疾走・静寂） | なのバナナ2 ✅ |
| Phase 1 noteヘッダー・記事ビジュアル | 静謐・和・本物感 | 別ツール推奨 |
| Phase 3 インバウンド向け | ラグジュアリー・極ミニマル | 別ツール推奨 |

**ライダービジュアルの軸（提案）**
> 「一本道の先に、不動心がある。」
道が続き、遠くに山か海があり、どこかに石碑の影がある。

**なぜYESか**
- Rapha・Pas Normal Studios等のプレミアムサイクリングブランドの文法と一致
- 「グリット（泥臭さ）と美学の共存」はハイエンドライダーが求めるもの
- アンチグラビリティの「ストリート感と静寂の同居」がライダーの自己認識に重なる

**なぜ「条件付き」か**
- MIGAKI BASEのコアブランド（不動心・航海士・歴史ある建物）は「重厚・静謐・本物」の格を持つ
- 2つのトーンが混在するとブランドの格が揺れる
- ライダー向けのフロントエンド広告に限定して使うこと
