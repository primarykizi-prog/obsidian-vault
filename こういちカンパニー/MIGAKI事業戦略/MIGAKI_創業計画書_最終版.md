## ■ 2．創業の動機・背景

### 2-1．創業に至った経緯

申請者は看護師として26年間（救急看護を中心に）勤務してきた。長年、医療現場で「人の命を左右するアセスメント」に向き合ってきた経験が、事業設計の根幹にある。

2026年、新潟県糸魚川市にある実家物件（元民宿・2棟・収容50名）が未活用のまま維持されていることに改めて向き合った。物件には以下の強みがあった。

- 設計士が設計した正規図面が存在する
- 消防設備が毎年点検されており基準を満たしている
- シーサイドバレー糸魚川スキー場から車で約1分の立地
- 日本海を望む唯一性のある景観
- 観光バス3台分の駐車場（50名収容）
- 借金ゼロ・両親が現地在住

「普通の民泊物件と出発点が違う。元民宿であるこの物件は、旅館業申請において最も有利な部類だ」という確信が、創業の決断につながった。

### 2-2．市場機会の認識

- インバウンド（訪日外国人）需要が全国的に拡大しており、地方の「本物の体験」への需要が急増している
- 糸魚川市は翡翠・ジオパーク・スキーを擁する複合観光地として注目が高まっている
- シーサイドバレーモトクロスコースに隣接しており、オフロードバイク層という固定需要も存在する
- AI技術の発展により、管理会社へ委託せず自社でOTA管理・多言語対応が可能な時代になった

### 2-3．ブランド名「MIGAKI」の確定

2026年4月2日、事業ブランド名を「MIGAKI（ミガキ）」と確定した。この名称には3つの意味を重ねている。

| 意味 | 訴求対象 |
|------|----------|
| 刀を磨く（日本文化・職人精神） | インバウンド・訪日外国人 |
| スキルを磨く（看護師の新しい働き方） | 都市部の看護師・医療従事者 |
| 自然の中で磨かれる（糸魚川の山・海・四季） | ライダー・ワーケーション層・移住検討者 |

## ■ 4．市場・競合環境の分析

### 4-1．市場環境

| 観点 | 内容 |
|------|------|
| インバウンド需要 | 訪日外国人数が回復・拡大中。地方の「本物の体験」への需要が特に増加 |
| 糸魚川の観光資源 | 翡翠・ジオパーク・スキー・海と山の共存という希少なロケーション |
| バイク需要 | シーサイドバレーモトクロスコース・風巻神社という地元ライダーの聖地が隣接 |
| 看護師不足 | 糸魚川・上越市は看護師人材が不足しており、行政課題となっている |

### 4-2．競合状況

糸魚川市内の民泊・旅館は限られており、以下の点で競合が存在しない領域がある。

- 50名収容×観光バス3台分の駐車場に対応できる施設は市内に希少
- インバウンド向け1棟丸ごと貸切に対応できる施設がない
- 「医療知識を持つ宿主」による安心感・ホスピタリティは他に存在しない

### 4-3．競争優位性

1. 元民宿の設備（設計士設計・消防設備点検済み）→ 許可取得コストが最小
2. スキー場から車1分・海と山を同時に望む唯一の立地
3. 50名収容・観光バス3台分の駐車場（大型グループ対応可能）
4. 借金ゼロ・維持費低水準（年間約10万円の固定資産税のみ）
5. 両親が現地在住・現地責任者として機能
6. AIエージェントによる管理会社機能の内製化（外注手数料20〜35%の削減）
7. 看護師26年・発信者としての信頼・1,200名の読者コミュニティが集客力になる

### 4-4．リスクと対策

| リスク | 対策 |
|--------|------|
| シーズン需要の偏り | インバウンド・バイク・合宿・ワーケーションで四季を埋める多層ターゲット |
| 両親の体力・健康問題 | シルバー人材センター活用・段階的に現地業務を外出し化 |
| 旅館業許可取得の遅延 | 元民宿物件のため通過確率が高い。保健所（志賀様）と連絡継続中 |
| 建物の老朽化 | 初期整備費を補助金で手当て。段階的にリノベーション |

本事業のブランド名「MIGAKI（磨き）」は、2026年4月2日に確定した。以下3つの意味を重ねた名称である。

|----|------|--------|

| 自然の中で磨かれる | 糸魚川の山・海・四季に鍛えられる | ライダー・ワーケーション層・移住検討者 |

```
MIGAKI BASE（旅館業）

MIGAKI NURSE（看護師マッチング）
```

- 管理会社を使わずAIで内製化するため、手数料20〜35%が手取りになる
- 借金ゼロの物件×補助金活用で、初期投資リスクが極めて低い
- 両親という現地責任者がいる → 申請者が現地常駐しなくても成立する

### 「旅館業法と民泊の制度的違い：365日営業には簡易宿所が正解」
- 民泊（住宅宿泊事業法）は届出だけで最短2〜3週間で開業できるが年間180日制限あり。簡易宿所（旅館業法）は許可制で審査に最短2〜3カ月かかるが365日営業が可能
- 2026年時点でも自治体条例による上乗せ規制があるため、糸魚川市の担当窓口への事前相談が必須
- 10人未満の施設は客室面積1人3.3㎡×人数以上で許可基準を満たせる（2016年改正で緩和済み）
- → **MIGAKIが選んだ旅館業法の簡易宿所は365日営業の観点から正解。開業目標から逆算して申請タイミングを設定する**
