## ■ 2．創業の動機・背景

### 2-1．創業に至った経緯

申請者は看護師として26年間（救急看護を中心に）勤務してきた。長年、医療現場で「人の命を左右するアセスメント」に向き合ってきた経験が、事業設計の根幹にある。

2026年、新潟県糸魚川市にある実家物件（元民宿・2棟・収容50名）が未活用のまま維持されていることに改めて向き合った。物件には以下の強みがあった。

- 設計士が設計した正規図面が存在する
- 消防設備が毎年点検されており基準を満たしている
- シーサイドバレー糸魚川スキー場から車で約1分の立地
- 日本海を望む唯一性のある景観
- 観光バス3台分の駐車場（50名収容）
- 借金ゼロ・両親が現地在住
- **日本庭園（鯉の池・巨岩・雪山）と「不動心」石碑** ← 2026-04-05判明

「普通の民泊物件と出発点が違う。元民宿であるこの物件は、旅館業申請において最も有利な部類だ」という確信が、創業の決断につながった。

### 2-2．市場機会の認識

- インバウンド（訪日外国人）需要が全国的に拡大しており、地方の「本物の体験」への需要が急増している
- 糸魚川市はフォッサマグナ・翡翠・ジオパーク・スキーを擁する複合観光地として注目が高まっている
- 「空間のゆとり（13㎡以上）が精神的健康を担保する」という学術的根拠のもと、50人収容を8名で使う高付加価値戦略が成立する
- AI技術の発展により、管理会社へ委託せず自社でOTA管理・多言語対応が可能な時代になった
- 同じ新潟県内の「浮遊のいえ（上越市）」が4名・1泊4万円で成立している実績から、MIGAKI BASEの5〜8万円帯設定は現実的射程内

MIGAKI BASEは「宿」ではなく「人生の道場（Luxury Dojo）」だ。世界を旅した航海士の父が「不動心」の石碑を置いた庭。剣道師範として磨き続けた人間が守る場所。そこへ看護師の息子が宿を作った。この三層の物語は、世界中の民泊で他に存在しない。

## ■ 4．市場・競合環境の分析

### 4-1．市場環境

| 観点 | 内容 |
|------|------|
| インバウンド需要 | 訪日外国人数が回復・拡大中。地方の「本物の体験」への需要が特に増加 |
| 糸魚川の観光資源 | フォッサマグナ（日本列島の誕生地）・翡翠（縄文6,000年）・スキー・海と山の共存という世界的に希少なロケーション |
| 武道・禅ツーリズム | 「Luxury Dojo」「Fudoshin」「Zen Garden」への富裕層インバウンド需要が拡大中 |
| 看護師不足 | 糸魚川・上越市は看護師人材が不足しており、行政課題となっている |

### 4-2．競合状況と差別化

ベンチマーク：浮遊のいえ（上越市）

| 項目 | 浮遊のいえ | MIGAKI BASE |
|---|---|---|
| 収容 | 最大4名 | **最大8名** |
| 料金 | 1泊4万円〜 | **目標：5〜8万円帯** |
| 事業形態 | 民泊新法（180日制限） | **旅館業法（365日）** |
| 所有権 | 不明 | **相続物件・所有権あり** |
| AI活用 | なし | **AIハイブリッド運営** |
| ホストストーリー | 不明 | **世界を旅した航海士×剣道師範×看護師** |

同じ新潟雪国で1泊4万円が成立している。MIGAKI BASEは規模・強みで**5〜8万円帯**が射程圏内。

### 4-3．競争優位性（2026-04-05最新版）

1. **不動心の庭園（最大の差別化）**：巨岩・鯉の池・「不動心」石碑・雪山が一画角に収まる。白馬の高級施設にない本物の日本庭園
2. **Luxury Dojoブランド**：航海士×剣道師範×看護師の三層。世界中の民泊で他に存在しない組み合わせ
3. **所有権あり・借金ゼロ**：相続物件。管理会社手数料20〜35%が丸ごと手取り。学術的にも「所有権ベース民泊が最も持続可能」
4. **元民宿の設備基準**：設計士設計・消防設備点検済み。旅館業申請で最も有利な出発点
5. **50人収容を8名で使う贅沢**：空間のゆとりが高単価を正当化（学術根拠：13㎡以上で精神的健康を担保）
6. **看護師オーナーの安心設計**：AED設置・医療グレード清掃・3言語緊急体制。「The host is a nurse. I knew my kids were safe.」
7. **AIハイブリッド運営**：こういちカンパニー（AIエージェント）が管理会社機能を内製化
8. **スキー場温泉（車1分）**：浴室設置義務免除の可能性あり（保健所確認中）
9. **365日対応**：旅館業法。白馬型長期滞在（1〜2週間）に完全対応

### 4-4．リスクと対策

| リスク | 対策 |
|--------|------|
| シーズン需要の偏り | インバウンド・バイク・合宿・ワーケーションで四季を埋める多層ターゲット |
| 両親の体力・健康問題 | 役割を「庭の番人」（鯉の世話・除雪）に限定。重労働は外出し化 |
| 旅館業許可取得の遅延 | 元民宿物件のため通過確率が高い。保健所（志賀様）と連絡継続中 |
| 建物の老朽化 | 初期整備費を補助金で手当て。キッチンリノベを最優先に段階的改修 |
