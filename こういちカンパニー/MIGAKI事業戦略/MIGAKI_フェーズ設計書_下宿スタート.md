# MIGAKI フェーズ設計書：下宿スタートからインバウンドへ

> **コンセプト**：完成してから開けない。粗くていい。泊まりながら磨く。それがMIGAKIだ。

---

## 🗺️ 全体像

```
Phase 0（現在）         Phase 1（下宿スタート）        Phase 2（インバウンド開始）
旅館業許可申請中   →   1〜2名の長期入居者受入   →   整備完了・外国人観光客へ
                        月7〜10万 × 1〜2名              高単価・一棟貸し
                        収入で少しずつ改装              Airbnb / Booking.com
                        入居者がMIGAKIを育てる
```

---

## Phase 0：現在地（2026年4月）

| 状態 | 内容 |
|------|------|
| 旅館業許可 | 申請中（保健所との往復完了・許可見込みあり） |
| 施設状態 | エアコン・トイレ整備があれば使用可能 |
| 初期改装 | まだ着手していない |
| 収入 | ゼロ |
| 次の一手 | Phase 1の設計を固め、最初の入居者を招く |

## Phase 1：下宿スタート

### 基本設計

| 項目 | 内容 |
|------|------|
| 受入人数 | 1〜2名（最初は1名から） |
| 家賃 | 月7〜10万円/人 |
| 入居期間 | 3〜6ヶ月〜（長期歓迎） |
| 入居スペース | 離れを優先活用（本館は両親が居住） |
| 契約形態 | 賃貸借契約（民泊ではなく長期滞在）または旅館業での長期滞在プラン |

### Phase 1の月次収支イメージ

```
収入（1名入居）  ：月7〜10万円
収入（2名入居）  ：月14〜20万円

固定費（光熱・固定資産税等）：月1〜2万円

→ 月5〜18万円が「改装積立」へ回せる
```

### 収入の使い道：改装優先順位

Phase 1収入は全額、Phase 2に向けた整備に充てる。

| 優先度 | 改装・整備項目 | 目安費用 | 理由 |
|:------:|--------------|---------|------|
| ★★★ | エアコン設置（客室） | 20〜40万 | 夏冬の快適性・必須 |
| ★★★ | トイレ・水回りの整備 | 20〜50万 | インバウンドの最低基準 |
| ★★☆ | Wi-Fi（Starlink導入） | 初期5万＋月1万 | ワーケーション・外国人必須 |
| ★★☆ | 清掃・寝具の整備 | 10〜20万 | Airbnb評価の根幹 |
| ★☆☆ | 共有スペースの整理 | 5〜15万 | 居心地の改善 |
| ★☆☆ | 外観・入口の整備 | 5〜10万 | 第一印象・写真映え |

> **Phase 2開始の基準**：上記★★★が全て揃い、Airbnbに掲載できる写真が撮れる状態になったとき。

## Phase 1入居者：こんな人に来てほしい

### ターゲット像

```
✅ 自然・静寂・雪・海を愛する人
✅ リモートワーカー・フリーランス・副業中の会社員
✅ 「何かを作ろうとしている」人（物書き・起業準備・自分探し）
✅ noteやSNSで発信している人（共創の楽しさを理解できる）
✅ 長期滞在OK・共有スペースの使い方を考えられる人
✅ 宿の成長に関わることを楽しめる人
```

### 招かない人

```
❌ 即座の快適さを求める人（まだ粗削りなため）
❌ 宿はサービスを買う場所だと思っている人
❌ 「なぜこんなに設備が古いの」と思う人
```

### 入居者リクルートの動線

```
1. note記事で「下宿入居者を募集する」記事を書く
2. こういちの読者（1,260人・INFJや看護師・自由を探す人が多い）に届ける
3. X投稿でシェア
4. DM・コメントで希望者と対話 → 入居決定
```

> noteの読者はすでに「こういちの哲学」に共感している人たち。
> 「モノを買う客」ではなく「一緒に育てたい人」が集まりやすい環境がある。

## 入居者を「共創者」として活用する設計

下宿入居者は単なる家賃収入源ではない。彼ら・彼女らがMIGAKIを磨く存在になる。

### 入居者に期待すること（任意・押しつけない）

| 役割 | 具体的なアクション |
|------|-----------------|
| フィードバッカー | 月1回、住んでみて感じたことをシェア（口頭orフォーム） |
| 発信者 | SNSやblogで糸魚川・MIGAKIの体験を発信（任意） |
| 協力者 | 簡単な改装DIYに参加（希望者のみ） |
| 紹介者 | 「次に来てほしい人」を紹介してくれることがある |

### フィードバックループの設計

```
入居者の声 → こういちが受け取る → 改装・整備に反映
→ 「あなたの声でここを変えました」と次の募集記事で報告
→ 次の入居希望者に「ここは住んだ人が育てている宿だ」が伝わる
```

> このループが「下宿」を「MIGAKI（磨き）」の体験に変える。

## Phase 1 → Phase 2 の移行判断基準

以下が全て揃ったとき、インバウンド受入を開始する。

```
□ エアコン設置済み（客室）
□ トイレ・水回り整備済み
□ Starlink（Wi-Fi）導入済み
□ 寝具・清掃体制が整っている
□ Airbnb掲載用の写真が撮れる状態
□ 両親が旅館業のチェックイン対応に慣れている
□ 旅館業許可の正式取得済み
```

**全て揃う目安**：Phase 1開始から1〜2年（収入積立ペースによる）

## Phase 2：インバウンド受入（参考）

詳細は [[MIGAKI_総合事業計画書]] および [[110_Riders_Luxury_Strategy]] を参照。

### 基本方針

- **ターゲット**：白馬スピルオーバーの欧米・オーストラリア系スキーヤー、ワーケーション層
- **単価**：1泊5〜10万円（一棟貸し）
- **チャネル**：Airbnb・Booking.com（多言語対応）
- **体験設計**：Quiet Luxury・Zero Friction・フォッサマグナ×根知男山ディナー

### Phase 1からの資産継承

| Phase 1で得たもの | Phase 2での活用 |
|----------------|--------------|
| 整備済みの客室・水回り | そのままAirbnb掲載 |
| 入居者のフィードバック | Airbnbリスティングの言語化に活用 |
| 入居者のSNS発信 | 口コミ・初期レビュー獲得の土台 |
| 両親の対応スキル | チェックイン・鍵渡しをスムーズに |
| こういちのnoteブランド | 「物語ある宿」として差別化 |
