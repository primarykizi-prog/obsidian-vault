### ① 中小企業新事業進出補助金（第4回）🔑 最重要

**主管：中小企業庁（経済産業省）**

| 項目 | 内容 |
|---|---|
| **公募期間** | 2026年5月19日〜（約1ヶ月） |
| **補助上限** | 大型（要公募要領確認） |
| **補助率** | 1/2以上（最低投資1,500万円が目安） |
| **対象業種** | **旅館業を明示的に含む** |
| **適用基準** | 資本金5,000万円以下 または 従業員200人以下 |
| **重点評価軸** | 「稼ぐ力の強化」「新事業への挑戦」|

**MIGAKI的活用ポイント：**
- 「古民家民泊」ではなく「インバウンド向け高付加価値宿泊事業への新規参入」として申請
- 「看護師26年の経験×不動心の庭×AIバックオフィス」というユニークな事業性が強み
- 1,500万円以上の投資計画が必要 → キッチン全面リノベ＋断熱＋乾燥室＋内装で十分到達

**経費配分の黄金比（審査通過の鍵）：**
| 費目 | 上限・指針 |
|---|---|
| 専門家謝礼 | 100万円上限 |
| 外注費 | 補助額の10%以内 |
| 広告宣伝費 | 売上見込みの5% |
| 設備・改修費 | **残り全額→ここを最大化** |

### ② 新潟県「稼ぐ力」強化設備投資支援 🔑 最大規模

**主管：新潟県（4月中旬募集開始予定）**

| 項目 | 内容 |
|---|---|
| **公募時期** | **2026年4月中旬〜（今すぐ情報収集）** |
| **補助上限** | **最大2,500万円** |

| **採択の鍵** | 賃上げへのコミットメント＋パートナーシップ構築宣言 |
| **対象経費** | 設備投資・商品開発・DX推進 |

**MIGAKI的活用ポイント：**
- 「インバウンド向けDX宿泊（AIバックオフィス＋スマートロック）」がDX要件に合致
- 両親への報酬設計（月次固定＋組数歩合）が「賃上げ・地元雇用」として機能
- キッチンリノベ＋断熱＋乾燥室＋スマート設備 → 単体で2,500万円枠を満たせる可能性
- **パートナーシップ構築宣言：** 地元施工業者・糸魚川商工会議所との連携が加点


## 🔑 採択を勝ち取る「必殺フレーミング」

### 申請書に必ず入れる言葉

```
❌ 「空き家を民泊として活用する」（採択されにくい）

✅ 「元民宿の建物を、インバウンド向け高付加価値宿泊事業として再生。
   AIバックオフィスと地元人材（両親）を組み合わせた
   DX×日本文化体験モデルで、糸魚川市の稼ぐ力を強化する新事業。」
```

### 事業性をアピールする3つの武器

1. **不動心の庭園**——白馬にない「本物の日本精神体験」が差別化軸
2. **AIバックオフィス**——小規模でも高単価を維持できる省力化の実証
3. **看護師26年の安心設計**——高齢者・障害者・家族連れにも対応できる施設安全性

## 📌 NICO「経営革新計画」について

新潟県産業創造機構（NICO）の「経営革新計画」承認を受けると：
- 各種補助金の採択優遇がある
- 低利融資（経営革新支援融資）の対象になる
- 「計画性のある事業主」として行政窓口での信頼度が上がる

**推奨：5月の申請前に、NICOへの相談を入れる。**
→ 商工会議所経由で無料相談が可能（糸魚川商工会議所）
