# MIGAKIプロジェクト｜総合事業計画書

> **看護師26年のキャリア・糸魚川の物件・AIの力。3つが交差する場所に、MIGAKIは立っている。**

---

## ビジョン宣言

49歳の看護師が、26年間磨き続けたアセスメント力と行動力を引っ提げて、日本海の見えるスキー場の麓に旗を立てる。
借金ゼロの物件がある。設計士が設計した建物がある。消防設備は毎年点検済みだ。
武器はすでに揃っている。あとは動くだけだった。

MIGAKIは「磨く」ブランドだ。
刀を磨く職人の姿勢で、看護師が自分を磨く場所で、糸魚川の自然の中で人が磨かれる。
旅館業で外貨を稼ぎ、マッチング事業で看護師の人生を変え、発信で日本中の仲間をつくる。
52歳のとき、こういちとその妻は、二人で設計した人生の中に立っている。

---

## なぜMIGAKIか

### ブランドネームの3層構造

| 層 | 意味 | 訴求先 |
|---|---|---|
| 刀を磨く | 日本文化・職人精神・研ぎ澄まされた技 | インバウンド・訪日外国人 |
| スキルを磨く | 看護師が田舎で自分を磨く体験 | 都市部の看護師・医療従事者 |
| 自然の中で磨かれる | 糸魚川の山・海・四季に鍛えられる | ライダー・ワーケーション層・移住検討者 |

### 糸魚川を選んだ理由

糸魚川は「海が見えるスキー場」という世界的に希少なロケーションを持つ。
シーサイドバレー糸魚川スキー場まで車で1分。観光バス3台分の駐車場。50人収容の2棟物件。借金ゼロ。

これは物件ではない。**インフラだ。**

インバウンドの波は来ている。田舎の看護師不足は深刻だ。発信に飢えた看護師は日本中にいる。
MIGAKIはその3つの波を、一つのブランドで受け止める。

## 事業3本柱

### 柱1｜MIGAKI BASE（旅館業）

#### 概要

| 項目 | 内容 |
|---|---|
| 形態 | 旅館業法・簡易宿所（保健所申請中） |
| 物件 | 糸魚川市・元民宿・2棟・50人収容 |
| 営業日数 | 365日（民泊新法の180日制限を受けない） |
| 立地優位性 | シーサイドバレースキー場まで車1分・海と山が同時に見える |
| 物件の強み | 設計士による正規図面・消防設備点検済み・旅館業基準を元々満たす |

#### ターゲット設計

**メインターゲット：インバウンド（1棟丸ごと貸切・高単価）**
- 訪日外国人のグループ旅行・スキー旅行
- 「MIGAKI」ブランドが刺さる日本文化体験層
- Airbnb・Booking.comでの多言語リスティング

**サブターゲット：バイク乗り（ライダーズベース・週末）**
- オフロードバイク乗り（シーサイドバレーモトクロスコース隣接）
- 風巻神社との連携で「バイクの聖地」直結の宿として口コミ集客
- 直接予約型・チラシ設置・SNS集客

**第三ターゲット：スポーツ合宿・ワーケーション**
- 50人収容×観光バス3台分の駐車場 → 合宿ドットコムでのB2B営業
- リモートワーカー・長期滞在ニーズへの対応

#### 運営体制

こういちカンパニー（AIエージェントチーム）が管理会社機能を内製化する。
外部管理会社への委託は不要。手数料20〜35%が丸ごと手取りになる。

### 運営体制（2026-04-05 更新：夫婦主体モデルへ）

**3層構造**：フロント（AI自動化）× バックエンド（こういち夫婦）× ローカル（両親・裏方プロ）

| レイヤー | 業務 | 担当 |
|---|---|---|
| **フロント（自動化）** | 予約管理・OTA対応・多言語メッセージ・スマートロック | AIエージェントチーム |
| **フロント（自動化）** | 価格設定・レビュー管理・チェックイン案内送信 | AIエージェントチーム |
| **バックエンド（経営）** | 意思決定・戦略・遠隔モニタリング・業者指示 | こういち＋妻（夫婦主体） |
| **バックエンド（経営）** | 清掃業者・メンテ業者の手配・評価 | こういち＋妻（デジタル管理） |
| **ローカル（裏方）** | 業者入室後の最終チェック・雪下ろし・敷地維持 | 両親（現場守り神） |
| **ローカル（裏方）** | 異常時の一次対応（設備故障・緊急連絡） | 両親（緊急時のみ） |

> ⚠️ **重要方針（2026-04-05確定）**：両親はゲストの前に姿を出さない。1泊数万円を払うインバウンド層は「プライバシー」と「プロのサービス」を求めている。アットホームすぎる演出は単価を下げる要因になる。

### 訪問頻度（2026-04-05確定）

**こういち夫婦：週1〜2回 糸魚川訪問可能**

→ これは「遠隔経営」ではなく「半現地経営」モデルを意味する。

| 訪問でできること | 内容 |
|---|---|
| 清掃・ターンオーバー | 業者委託なしで夫婦が自前対応できる（繁忙期のみ外注） |
| 品質チェック | 毎ターンオーバー後に直接確認 → 5星品質の維持 |
| DIYリノベ | 週1〜2回の積み上げで内装・照明・家具を自分たちで完成させる |
| 両親サポート | 定期的に顔を出すことで両親の安心感・モチベーションを維持 |
| 現場モニタリング | 遠隔カメラ＋週次訪問で「見えない問題」を早期発見 |

> **清掃コスト試算**：外注の場合1回あたり15,000〜30,000円。週1〜2回訪問なら自前対応でほぼゼロ。年間100〜200万円のコスト差になる可能性あり。

### 両親へのモチベーション設計
- **毎月の収益報告**：数字で「実家が稼いでいる」を可視化
- **ゲストの感謝の声**（AI日本語訳）：「世界のスキーヤーに絶賛された実家」を両親に共有
- **IT活用**：スマートロック・清掃管理アプリの操作習得 → 生きがい＋資産価値向上に直結

### 柱2｜MIGAKI NURSE（看護師マッチング事業）

#### コンセプト

> **都会のナース × 糸魚川・上越の病院・施設 × MIGAKI BASEに宿泊**
> 田舎で働きながら自分を磨く看護師のためのプラットフォーム

#### なぜこういちにしかできないか

- note読者 = 看護師の働き方に悩む人。その読者がそのままユーザーになる
- 26年の現場経験・救急看護が生む「看護師目線のキュレーション」
- 発信者としての信頼（MIGAKI MEDIAとの一体構造）
- 旅館業との一体化で「泊まりながら働く」体験を提供できる

#### 参入戦略（段階的）

```
Phase1：noteの延長で無料マッチングスタート
　　↓　（有料職業紹介許可は不要な範囲でスタート・リスクゼロ）
Phase2：コミュニティ形成（Xメンバーシップ・noteサークル）
　　↓
Phase3：規模が証明できれば有料人材紹介へ拡張
　　↓
Phase4：AIエージェントがコーディネートを担うWebアプリ化
```

#### 競合優位性

| 比較軸 | 既存サービス | MIGAKI NURSE |
|---|---|---|
| ターゲット | 給料・条件重視の転職者 | ライフスタイル重視の看護師 |
| 手法 | 求人データベース型 | コミュニティ型 |
| 場所 | 都市部中心 | 田舎・地方特化 |
| 宿泊 | なし | MIGAKI BASEと一体化 |
| 信頼の源泉 | 企業ブランド | 現場26年の看護師CEO |

**田舎 × ライフスタイル重視 × コミュニティ型は誰もやっていない。競合の空白地帯だ。**

#### 行政連携の可能性

- 看護師不足は糸魚川・上越市の行政課題でもある
- MIGAKIが地域課題の解決手段になれば、補助金・行政協力が引き寄せられる
- 民泊の未来会議連携・商工会議所との関係を延長線上で活用できる

## 強みとリスク分析

### 強み（なぜ勝てるのか）

| 強み | 詳細 |
|---|---|
| 借金ゼロの物件 | 初期負債なし。キャッシュフローがそのまま手取りになる |
| 元民宿の設備 | 設計士による正規図面・消防設備点検済み・旅館業基準を元々満たす可能性が高い |
| 立地の希少性 | 海が見えるスキー場・モトクロスコース隣接・観光バス3台分の駐車場 |
| インバウンドの波 | 訪日外国人が増え続けている。地方の本物の体験への需要は拡大中 |
| 発信者としての信頼 | 26年の現場経験・note読者との接点がそのまま集客力になる |
| AIチーム内製化 | 管理会社手数料（売上の20〜35%）を丸ごと内製化できる |
| 夫婦の役割分担 | こういちの行動力×妻の財務・デザイン力。二人で事業を設計している |
| 行政との関係構築 | 保健所・市役所・商工会議所・未来会議・地域との連携が始まっている |

### リスクと対策

| リスク | 影響度 | 対策 |
|---|---|---|
| 旅館業許可取得の遅延 | 中 | 保健所（志賀様）との連絡を継続。元民宿物件なので通過確率は高い |
| オフシーズンの稼働率低下 | 高 | インバウンド×合宿×ワーケーションで季節を埋める多層ターゲット |
| 両親の体力・健康問題 | 高 | シルバー人材センター活用・段階的に現地対応を外出し化 |
| 競合施設の増加 | 低〜中 | MIGAKIブランド・発信者コミュニティ・看護師ネットワークでの差別化 |
| 看護師業務との両立 | 中 | AIチームで運営を自動化。月1〜2時間の確認のみで回る設計 |
| マッチング事業の法規制 | 低 | 無料マッチングでスタート。有料人材紹介は規模証明後に検討 |

### 「簡易宿所許可申請は2〜3カ月・開業目標から逆算せよ」
- 民泊（住宅宿泊事業法）と簡易宿所（旅館業法）は法律・手続きが別物。簡易宿所は許可制で審査があり最短2〜3カ月かかる
- 365日営業を目指すなら簡易宿所が正解（民泊新法の180日制限を受けない）
- 10人未満の施設は客室面積1人3.3㎡×人数以上で許可基準を満たせる（2016年改正で緩和済み）
- → **開業目標日から逆算して許可申請スケジュールを設計する。糸魚川市担当窓口への早期事前相談が最初のアクション**
- 出典：民泊・旅館業の3つの枠組み解説（LINKEY）/ 簡易宿所営業許可完全ガイド（行政書士中尾事務所）/ 旅館業法許可徹底ガイド（tabilmo）・2026-04-03 digest
