# 🔨 MIGAKI工事・設備 補助金対象使い分けチェックリスト

> **大原則：発注・工事・支払いはすべて「交付決定通知」の後。**
> 採択≠交付決定（採択後にさらに交付申請が必要）。
> 工事着手の最短ラインは採択発表（第3回なら2026年7月頃）のさらに後。

---

## 🟢 今すぐ自費でやってOK（補助金対象にしない）

| チェック | 項目 | 理由 |
|---|---|---|
| ☐ | 消防設備の整備（火災報知器・誘導灯・消火器等） | 旅館業許可の法的義務。待てない。補助金対象外が多い |
| ☐ | 網戸・窓ガラスの軽微な修繕 | 少額。補助金申請するほどでもない |
| ☐ | 清掃用品・洗剤・備品類（少額消耗品） | 消耗品は基本的に補助対象外 |
| ☐ | 4/20消防現地確認のための整理・片付け | 申請前準備であり経費計上できない |
| ☐ | 両親が使う日常的な生活修繕 | 事業と区別できないものは対象外 |

## ⚠️ 申請前に絶対やってはいけないこと

```
❌ 補助金申請前に工事を発注・着工・支払いする
❌ 採択通知が来た段階で「もう大丈夫」と思って発注する（交付決定が必要）
❌ 補助金で取りたい工事を先に自費でやり「後で精算」しようとする（不可）
❌ 見積書なしで工事を進める（屋号入り正式見積書が必須）
❌ Before写真を撮り忘れる（工事前の現状写真がないと補助金が下りない）
```

## 📅 工事着手のタイムライン（第4回・創業型を狙う場合）

```
2026年6〜7月  → 糸魚川創成塾2026 受講
2026年7月末   → 特定創業支援等事業の証明書 取得
2026年夏      → 開業届 提出（今井さんと相談してタイミング決定）
2026年夏秋    → じゅん君に「見積書だけ」先に依頼しておく（発注はまだ）
               → Before写真を撮影しておく（工事前の記録）
2027年春（予測）→ 創業型 第4回 申請
2027年夏（予測）→ 採択発表 → 交付申請 → 交付決定
2027年夏以降  → ✅ ここからようやく工事発注・着工・支払いOK
```
