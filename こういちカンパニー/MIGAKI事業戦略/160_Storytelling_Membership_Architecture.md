# 🧭 160_Storytelling_Membership_Architecture
## MIGAKI BASE メンバーシップ設計図
### 「プロセスを共有することが、最高の集客になる」

> **この設計書が答える問い：**
> 「なぜ、お金を払う人が、同時にMIGAKIの成長を助けるのか。」
>
> **答え：私たちは"結果"を売らない。"過程"を売る。**
> 行政との格闘。補助金の壁。工事の失敗。それをリアルタイムで共有することが
> ITワーカーの心を掴む最強のコンテンツになる。

## 🧭 航海士プラン（SUPPORTER層）

### コンセプト：「見届ける人が、応援団になる」

> IT業界には「Building in Public（公開しながら作る）」という文化がある。
> GitHubのコミット履歴を公開するように。失敗も成功も実況するように。
> こういちが保健所・消防・補助金と格闘する様子を「MIGAKI航海日誌」として公開することが
> ITワーカーに深く刺さる理由がここにある。

#### 価格：¥980/月（年払い：¥9,800/年）

#### 会員特典

| # | 特典 | 内容 | 頻度 |
|---|-----|------|------|
| 1 | **🗒 MIGAKI航海日誌** | 保健所・消防・補助金の格闘をリアルタイム実況。「今日、様式3を5回直された話」「保健所の担当者が変わった話」— こういちにしか書けないドキュメント | 週1〜2本 |
| 2 | **🎙 月1回「航海会議」** | オンラインで事業進捗を報告。Q&A形式で何でも聞ける。航海士だけの参加権 | 月1回 |
| 3 | **⏰ 先行予約権** | 宿泊・体験（富裕層ライダープラン・滞在）の一般公開72時間前に案内が届く | 随時 |
| 4 | **📜 第0期航海士の証** | MIGAKI BASEの歴史に名前が刻まれる（物件に小さなプレートを設置予定）| 一生涯 |

---

#### なぜ「航海日誌」がITワーカーに刺さるか（論理化）

```
ITエンジニアが「過程の公開」に熱狂する3つの理由：

① 教育価値
   「実際の起業家が行政とどう格闘するか」は
   ビジネス書には絶対に書いていない。
   1,000円の本より、月980円のリアル実況の方が価値がある。

② 共感・応援衝動
   困難に立ち向かう人間の物語は、
   Kickstarterやクラウドファンディングと同じ心理を刺激する。
   「この人を成功させたい」と感じた人は課金する。

③ 先行情報へのアクセス
   「住んでみたいかも」と思った人は
   先行予約権を使うために航海士になる。
   つまり航海士は住人の予備軍でもある。
```

---

#### Win-Win分析：航海士プラン

| 視点 | 価値 |
|------|------|
| **会員側** | 月980円で「実際に動いている起業の現場」を追いかけられる。IT転職・副業・地方移住を考えている人の参考事例として最高の教材 |
| **MIGAKI側** | 月980円 × 50名 = 月4.9万円の固定収入（記事を書くだけ）。かつ「ファンベース」の形成。住人・提督への転換率が高い層を無広告で育成できる |
| **エコシステム** | 航海士が記事を「シェア・引用」することで、無料記事のリーチが広がる → 新しい航海士が入る → 好循環 |

## 🏠 住人プラン（RESIDENT層）

### コンセプト：「場所を変えることが、人生のアップデートになる」

> 都会のアパートを解約して来る人は、
> 「どこかで何かを変えたい」と感じている。
> その「変わりたい」という衝動に、
> 看護師のアセスメント力とMIGAKIの環境が応える。

#### 価格

| コース | 月額 | 対象 |
|-------|------|------|
| **スタンダード（個室1室）** | ¥120,000〜150,000 | リモートワーカー・フリーランス |
| **プロ・リカバリー（個室2室）** | ¥300,000 | CTO・フリーランスコンサル・高収入層 |

→ 詳細設計は [[120_Workation_Strategy_Integrated]] を参照

#### 住人だけが手に入れるもの

| # | 特典 | 内容 |
|---|-----|------|
| 1 | **🛸 Starlink専有環境** | 爆速衛星インターネット。最大300Mbps。「コードが書ける」を保証する |
| 2 | **🪑 エルゴノミクス環境独占** | ハーマンミラー・スタンディングデスク・外部モニター完備のワークステーション |
| 3 | **🏥 週1 リカバリーブリーフィング** | こういちによる個別身体アセスメント（30分）。「バーンアウトの前兆を先読みして手を打つ」看護師的アプローチ |
| 4 | **🏔 フォッサマグナウォーク** | 月1回・世界的地質遺産を案内（夕方1.5時間）。大地の話をしながら自分の現在地を見直す |
| 5 | **📝 180日後の自分へのレター** | 入居初日に書いて封をする。退去日に受け取る。MIGAKI BASEが「変化の証人」になる |
| 6 | **📖 MIGAKI住人として記録される** | こういちのnote記事に「第1期住人」として登場する権利（希望者のみ）。これが次の住人を呼ぶ |

---

#### Win-Win分析：住人プラン

| 視点 | 価値 |
|------|------|
| **会員側** | 月12〜15万円で「都市と同等以上のインフラ + 看護師的リカバリーサポート + 圧倒的な自然環境」を得る。都市での月18.5万円の生活（家賃+食費+交通費+コワーク）より安く、豊かになれる |
| **MIGAKI側** | 月12〜30万円の安定収入。さらに住人の「体験談」がnoteの次の無料記事になり、次の住人を呼ぶ自己増殖型集客が完成する |
| **エコシステム** | 住人が書いたブログ・Xへの投稿が「MIGAKI BASEのUGC（ユーザー生成コンテンツ）」として機能し、口コミ集客に直結 |

## ⚓ 提督プラン（PARTNER層）

### コンセプト：「こういちカンパニーの共同設計者になる」

> 提督は「お客様」ではない。
> MIGAKI BASEという実験場に「一緒に乗り込む」人だ。
> 資金・スキル・人脈を持ち込み、共に事業を設計する。
> 見返りは、金銭以上の「実戦経験」と「未来の持分」だ。

#### 対象者

```
- テック系スタートアップのCEO・COO
- VC・エンジェル投資家（社会的インパクト案件として）
- 法人ウェルネス担当（RaaS契約として）
- 地方移住を検討している上場企業の役員
- MIGAKI BASEの「概念」に惚れた個人
```

#### 提督が得るもの

| # | 内容 | 価値 |
|---|-----|------|
| 1 | **こういちカンパニー 月2回の共創セッション** | 事業設計・課題解決のブレスト。「ゼロから事業を作る現場」に参加できる |
| 2 | **MIGAKI BASEの将来持分オプション** | 個別交渉。事業が軌道に乗った段階での出資・共同経営の優先交渉権 |
| 3 | **法人請求書対応** | 福利厚生費・研修費として計上可能。税務的に「経費」になる |
| 4 | **共著note記事** | 「提督として関わった話」をこういちと共著で発信。相手のブランディングにも使える |
| 5 | **RaaS枠（月定額宿泊利用）** | 月10泊分の優先利用権。年間スケジュールで社員を送り込める |

---

#### Win-Win分析：提督プラン

| 視点 | 価値 |
|------|------|
| **提督側** | 「日本の地方再生」「医療×観光×テック」という複数の市場が交差するユニークな実験場への参加権。単なるワーケーション施設への投資ではなく、「物語のある事業」への共創体験 |
| **MIGAKI側** | 資金・スキル・人脈の補完。補助金に詳しいITエンジニアが提督になれば申請が加速する。VCバックグラウンドを持つ提督は資金調達ルートを開く |
| **エコシステム** | 提督がMIGAKIを「自分の事業の一部」と感じることで、超強力なアンバサダーになる |

## 📅 今週末の発表スケジュール（金・土・日）

### 🗓 金曜｜序章：「安全への執念」

**タイトル案：**
> 「消防に2回、保健所に4回落とされた。それでも私が作り続ける理由。」

**内容構成：**
```
①【フック】「今日、保健所から電話があった」—— 最初の一文で引き込む
②【問い】なぜ看護師が宿を作るのか（26年の救急経験との関係）
③【格闘の実況】様式3を何回直したか。何を言われたか。
④【感情の開示】心が折れそうになった瞬間と、それでも続けた理由
⑤【核心】「26年間、安全基準を守ることで命を救ってきた。だから逆算できる」
⑥【次話予告】「明日、全部話す。父の話も含めて。」
```
**目的：** 「この人を応援したい」という感情を植え付ける

### 🗓 土曜｜本編：「MIGAKI BASEの全貌 + メンバーシップ発表」

**タイトル案：**
> 「父は甲種一等航海士だった。私は救急看護師だ。二人で『生きる場所』を作った。」

**内容構成：**
```
①【父の話】甲種一等航海士・2〜3年の外洋航海・不動心の石
②【物件の話】実家の元民宿が空になった日、何を感じたか
③【コンセプト開示】Zero Friction, Active & Zen — 何のために作るのか
④【設備の公開】Starlink・ハーマンミラー・フォッサマグナ断層
⑤【メンバーシップ発表】航海士 / 住人 / 提督の3層を初公開
⑥【Call to Action】「航海士になって、一緒に航海を見届けてください」
```
**目的：** 会員登録への具体的なアクションを促す

### 🗓 日曜｜募集：「第1期住人・3名限定招待状」

**タイトル案：**
> 「180日、モンク・モードで生きませんか。—— MIGAKI BASE 第1期住人・3名限定」

**内容構成：**
```
①【モンク・モードとは】雑音を断ち、一つのことに集中する期間
②【提供するもの全リスト】Starlink・エルゴ・看護師・断層・料金全込み
③【求める住人像】「3ヶ月後に何を完成させたいか」が答えられる人
④【応募条件】航海士会員（¥980/月）であること＋選考フォーム提出
⑤【選考の理由】「3名を厳選する。なぜなら最低ラインが天井を決めるから」
⑥【締切と開始日】◯月◯日締切 / ◯月◯日入居開始
```
**目的：** 「住みたい」と思った人を具体的なアクションに変換する
