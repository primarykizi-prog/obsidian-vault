# 📝 130_Note_Membership_Business_Model
## noteを起点にした「広告費ゼロ・満室」プラン

> **前提：** 既存の「MIGAKI道場」設計（白帯500円・青帯980円・黒帯3,000円）は残す。
> このファイルはその上に「ITワーカー長期滞在者の獲得」レイヤーを追加する。

## 🎯 集客の全体構造（3段階ファネル）

```
【Stage 1：拡散】無料note記事
         ↓ 共感した人が「いいね・シェア」で広がる
【Stage 2：選別】有料メンバーシップ（MIGAKI道場）
         ↓ 本気の人だけが課金する＝意欲の証明
【Stage 3：成約】メンバー限定の入居者募集
         ↓ 3名のみ・先着ではなく「選考」形式
【居住開始】MIGAKI BASE 180日間プログラム
```

**広告費：ゼロ。必要なのはこういちの文章だけ。**

## Stage 1｜拡散記事（無料）— 「伝説の1記事目」

### タイトル案（3つ）

**案A（最強・直球）：**
> 「冬が来る前の180日。都会のアパートを解約して、人生を変えるコードを書かないか？」

**案B（物語型・じわじわ刺さる）：**
> 「26年間、人が死ぬ場所で働いてきた私が、"人が生きる場所"を作った理由。」

**案C（問いかけ型・不安を刺激する）：**
> 「あなたのコードが止まっているのは、能力の問題じゃない。場所の問題だ。」

> 💡 **推奨：案Aをメインに、案Bのサブタイトルを添える形が最強。**
> 「冬が来る前の180日。〜コードを書かないか？—— 26年間、人が死ぬ場所で働いてきた看護師が作った、生きるための宿。」

### 📄 今日投稿すべきリード文｜冒頭300文字

> ⚠️ **このセクションが今日の最優先アウトプット。以下をそのままnoteに貼り付けてよい。**

```
コードが書けない日が、続いていた。

Slackの通知が来るたびに、胃が重くなる。
会議室でラップトップを開くたびに、呪いみたいに眠くなる。

問題は、仕事じゃない。場所だ。

今春、私たちは北アルプスの麓の元民宿を改装した。
Starlinkを引いた。ハーマンミラーを3台置いた。

あとは、本気で生き直したいエンジニアを
3人だけ、迎える。

冬が来るまでの180日間。
月12万円。家賃・光熱費・Wi-Fi・すべて込み。

都会のアパートを解約して、ここへ来なさい。
あなたが今、書けないでいるコードの続きを——
ここで書いてほしい。

                         ── こういち（49歳・看護師・MIGAKI CEO）
```

### 記事の構成案（全体）

| セクション | 内容 | 字数目安 |
|-----------|------|---------|
| **リード文** | 上記300文字（そのまま使う） | 300字 |
| **私は誰か** | 26年救急看護師・実家の元民宿・父の不動心 | 400字 |
| **なぜ今か** | バーンアウトの波・リモートワークの孤独・場所の力 | 500字 |
| **MIGAKIとは何か** | 物件スペック・Starlink・ハーマンミラー・フォッサマグナ | 600字 |
| **価格と条件** | 月12〜15万・3名限定・180日間・2026年春〜 | 300字 |
| **応募方法** | noteメンバー（黒帯）になってDMを送る | 200字 |
| **橋渡し一文** | 「続きはMIGAKI道場（有料）で」 | 100字 |

**合計：約2,400字。読了時間6〜7分。**

## Stage 2｜有料メンバーシップ（選別機能）

> **既存「MIGAKI道場」の構造を維持しつつ、ITワーカー獲得の「選別装置」として機能させる。**

### なぜ有料メンバーシップが「選別」になるか

```
月980円（青帯）を払う人＝
  ✅ MIGAKIに本気で関心がある
  ✅ 情報収集コストを払える（経済力の証明）
  ✅ 「乗っかる前に見極めたい」慎重さを持つ

月3,000円（黒帯）を払う人＝
  ✅ 泊まりたい意思がある（優先予約権が目当て）
  ✅ こういちを「応援したい」共感レベルの高い層
  ✅ → この層から入居者候補を探す
```

### 追加コンテンツ（ITワーカー向け・新設）

**青帯以上に追加するコンテンツ：**

| # | タイトル | 内容 |
|---|---------|------|
| 1 | 「Starlinkを糸魚川の山奥に引いた話」 | 機材・工事・実測速度・仕事への影響 |
| 2 | 「ハーマンミラーを3台置いた理由と、看護師的エルゴノミクス論」 | 姿勢・腰痛・疲労の医学的視点 |
| 3 | 「フォッサマグナの断層の上で、3000行のコードを書いた」 | 入居者体験談（第1期）|
| 4 | 「26年の救急看護師が考える、エンジニアのバーンアウト予防法」 | 医学的視点×ITワーカー |
| 5 | 「MIGAKI BASE 建設日誌#X：じゅん君がトイレを直した日」 | 泥臭いリアル・失敗談 |

## Stage 3｜メンバー限定募集（成約）

### 募集フロー（広告費ゼロ版）

```
Step 1：無料記事を公開
         ↓（1〜2週間でシェアが広がる）
Step 2：記事末尾に「MIGAKI道場（月980円〜）に入ってDMを」
         ↓（本気の人だけが課金して応募してくる）
Step 3：黒帯会員に「入居者選考」の案内記事を公開（有料・メンバー限定）
         ↓（応募フォームを公開）
Step 4：応募者の中から「Googleフォーム選考」
         ↓（質問：職種・なぜここを選んだか・3ヶ月で何を達成したいか）
Step 5：選考通過者に個別連絡・契約
         ↓
Step 6：入居開始 → 体験記事をnoteで公開 → 次の入居者が集まる（自己増殖）
```

### 選考フォームに入れる質問（案）

| # | 質問 | 意図 |
|---|-----|------|
| 1 | 「今の仕事と、リモートかどうか」 | 仕事できる環境か確認 |
| 2 | 「都会でいくらの家賃を払っているか」 | 支払能力・等価交換の理解度 |
| 3 | 「180日間でここに来て、何を作りたいか・何を変えたいか」 | 本気度・目的の明確さ |
| 4 | 「MIGAKI BASEを知ったきっかけ（記事・SNS・口コミ）」 | 集客チャネルの把握 |
| 5 | 「喫煙の有無・アレルギー・苦手なこと」 | 共同生活の適合性 |

### 「先着ではなく選考」にする理由

> **「3人だけ選ぶ」という設計が、価値を守る。**
>
> 先着順にすると「早く申し込んだ人」が来る。
> 選考にすると「MIGAKIに合う人」が来る。
>
> 合わない人が1人入ると、コミュニティ全体の質が下がる。
> こういちが26年で学んだこと：チームの最低ラインが、チームの天井を決める。

## 📊 売上シミュレーション（noteだけで）

```
無料記事 → 1,000PV → 50人がメンバーに
  白帯50人 × 500円 = 25,000円/月
  青帯20人 × 980円 = 19,600円/月
  黒帯 5人 × 3,000円 = 15,000円/月
────────────────────────────────
noteメンバーシップ月収：59,600円/月

+ 入居者3名（月12万×3名）：360,000円/月
────────────────────────────────
note × 入居者 合計：419,600円/月
（≒ 年間500万円・広告費ゼロ）
```

## 📅 [UPDATED: 2026-04-09] 今週末の3日間スケジュール（金・土・日）

> **メンバーシップ3層（航海士/住人/提督）の発表から住人募集までを3日間で完結させる。**
> 詳細なストーリーライン・各記事の構成 → [[160_Storytelling_Membership_Architecture]]

### 金曜｜序章「安全への執念」

**投稿タイトル（案）：**
> 「消防に2回、保健所に4回落とされた。それでも私が作り続ける理由。」

**狙い：** 「この人を応援したい」感情を植え付ける

**核心メッセージ：**
> 「26年間、安全基準を守ることで命を救ってきた。だから逆算できる。
> 普通の投資家には書けない申請書を、看護師は書ける。」

**末尾CTA：** 「明日、全部話します。父の話も含めて。」

---

### 土曜｜本編「MIGAKI全貌 + 航海士/住人/提督 発表」

**投稿タイトル（案）：**
> 「父は甲種一等航海士だった。私は救急看護師だ。二人で『生きる場所』を作った。」

**狙い：** メンバーシップ3層を初公開し、航海士（¥980/月）への登録を促す

**核心メッセージ：**
> 「私たちは"結果"を売らない。"過程"を売る。
> 保健所との格闘も、補助金の挫折も、すべてがコンテンツになる。
> それを一緒に見届けてくれる人を、航海士と呼ぶ。」

**末尾CTA：** 「まずは航海士（¥980/月）になってください。明日、住人募集の話をします。」

---

### 日曜｜募集「第1期住人・3名限定招待状」

**投稿タイトル（案）：**
> 「180日、モンク・モードで生きませんか。—— MIGAKI BASE 第1期住人・3名限定」

**狙い：** 航海士会員の中から、住人（月12〜15万）を3名選考する

**核心メッセージ：**
> 「先着ではなく、選考にします。
> 3名を厳選する。なぜなら、チームの最低ラインが天井を決めるから。
> 26年の救急現場で学んだことは、そういうことです。」

**応募条件：**
- 航海士会員（¥980/月）であること
- 選考フォーム（Googleフォーム）提出
