# 📁 Vault構造

```
10_自分OS/                        ← プロフィール・コアテーマ・状態管理・目標管理
  └ _claudian-state.md           ← ★セッション開始時に必ず読む・現在地管理
  └ _action-queue.md             ← 未処理タスク一覧
20_知識ベース/                    ← digest保存先・学習ログ
  └ YouTube学習/                 ← YouTube文字起こし学習ノート
  └ AI・ツール/                  ← AI活用・ツール系知識
  └ Web知見収集/                 ← digest保存先
30_記事製造ライン/                ← 執筆中・ネタ・アイデア・公開済み
50_夫婦・家族/                    ← 夫婦OS統合・ライフデザイン
こういちカンパニー/               ← MIGAKI事業全体
  └ CEO/                        ← CEO_CLAUDE.md（AI運用ルール）
  └ MIGAKI事業戦略/              ← 総合事業計画書・創業計画書・キーマンリスト
  └ 各部署（予約/財務/マーケ等）  ← MIGAKI BASE（旅館業）運営
日報/                             ← ★現役の日報（2026-04〜）Cowork日報はここ
60_日報/                          ← アーカイブ（〜2026-03）
秘書メモ/                         ← ★現役の秘書メモ（思いつき・アイデア・雑メモ）
60_日報/秘書メモ/                  ← アーカイブ（〜2026-03）
.claude/agents/                   ← サブエージェント定義
.claude/rules/                    ← 分割ルールファイル（このファイルもここ）
```

> **注意**：`日報/` と `60_日報/` は別物。現役は `日報/`（ルート直下）。
> 秘書メモも同様に `秘書メモ/`（ルート直下）が現役。`60_日報/秘書メモ/` はアーカイブ。
