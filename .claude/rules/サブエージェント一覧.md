# 🤖 サブエージェント一覧

| エージェント | 役割 | 起動コマンド |
|------------|------|------------|
| `digest-agent` | Web検索→Vault保存 | 「digest」 |
| `reflector-agent` | アクションメモ→ノート反映 | 「a」 |
| `writer-agent` | 構成案→記事本文執筆 | 「b」 |
| `planner-agent` | アイデア→記事構成案生成 | 「p」 |
| `session-agent` | セッション引き継ぎ更新 | 「s」 |
| `weekly-agent` | 週次知識統合レポート | 「weekly」 |
| `note-publisher-agent` | 記事→note.com投稿用変換 | 「n」 |
| `memory-agent` | 眠っている記憶を週1回発掘 | 「m」 |
| `secretary-agent` | 思考・アイデアを即Vault保存 | 「秘書」 |
| `x-post-agent` | X投稿案を曜日別に3案生成 | 「x」 |
| `editorial-agent` | 次に出す記事の順序を戦略的に決める | 「e」 |
| `reviewer-agent` | 執筆済み記事を品質チェック | writer-agent完了後に自動 |
| `system-monitor-agent` | チームの自己診断・自己改善 | weekly連鎖の末尾で自動 |
| `minpaku-agent` | 糸魚川民泊プロジェクト管理 | 「民泊」「糸魚川」 |
