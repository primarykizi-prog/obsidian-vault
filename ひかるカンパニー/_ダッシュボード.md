# 🏢 ひかるカンパニー ダッシュボード

> CEOであるひかるが「今うちの会社どうなってる？」と思ったらここを見る。

---

## 組織図

```
ひかる（CEO）
  │ 指令・判断・意思決定のみ
  │
  ▼
秘書（secretary）── 唯一の窓口・振り分け・組織管理
  │
  ├── blog（ブログ事業部）
  │   └ ナースXのカンファレンス室の記事制作・SEO・アフィリエイト
  │
  ├── note（note編集部）
  │   └ note記事の企画・執筆・リライト（2軸運営）
  │
  ├── research（リサーチ部）
  │   └ Web新着digest処理・トレンド調査・知識蓄積
  │
  ├── marketing（マーケティング部）
  │   └ X運用・SNS戦略・認知拡大
  │
  └── pm（PM）
      └ 進捗管理・タスク管理・優先順位づけ
```

## 情報の流れ

```
Web新着digest
  ↓
リサーチ部（3トピックス選定・知識保存）
  ↓
秘書（振り分け）
  ├→ noteネタ → note編集部
  ├→ ブログネタ → ブログ事業部
  └→ X投稿案 → マーケティング部
```

```
ひかるの思考・指令
  ↓
秘書（整理・分類・振り分け）
  ├→ 記事制作 → blog or note
  ├→ 調査依頼 → research
  ├→ SNS関連 → marketing
  └→ 進捗確認 → pm
```

## 経営方針

- 自分がやるのは「指令・判断・意思決定」のみ
- それ以外はAIエージェント（社員）に任せる
- AI社員同士で連携して仕事を進める
- 部署は必要になったら作る。いらない部署は作らない
- 知識・記録はすべてObsidian Vaultに蓄積する
