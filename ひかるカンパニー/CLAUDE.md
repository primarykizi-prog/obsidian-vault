# ひかるカンパニー

## このディレクトリについて

ここは「ひかるカンパニー」の本社。
CEOであるひかる（看護師26年目・経営情報学士）が運営するAIカンパニー。

## 起動ルール

このディレクトリ内のファイルに触れたら、まず以下を確認すること：

1. **自分がどの部署のエージェントとして動くか**を確認する
2. 該当する部署の `CLAUDE.md` を読み込む
3. そのCLAUDE.mdのルールに従って行動する

部署が不明な場合は、`secretary/CLAUDE.md`（秘書）を読み込む。
秘書が全部署への窓口・振り分けを担当している。

## 組織構成

```
ひかるカンパニー/
├── CLAUDE.md              ← このファイル（会社全体のルール）
├── _ダッシュボード.md      ← 全体像・組織図・情報の流れ
├── _デイリーオペレーション.md ← 毎日の使い方マニュアル
├── _ワークフロー.md         ← 勤務パターン別の手順書
│
├── CEO/CLAUDE.md          ← ひかるの経営方針
├── advisor/CLAUDE.md      ← 顧問（設計・評価・改善・週次レビュー）
├── secretary/CLAUDE.md    ← 秘書（唯一の窓口・振り分け・組織管理）
├── blog/CLAUDE.md         ← ブログ事業部（記事制作・SEO・AF）
├── note/CLAUDE.md         ← note編集部（企画・執筆・リライト）
├── research/CLAUDE.md     ← リサーチ部（digest処理・知識蓄積）
├── marketing/CLAUDE.md    ← マーケティング部（X運用・SNS戦略）
└── pm/CLAUDE.md           ← PM（進捗管理・タスク管理）
```

## 基本原則

- CEOの指令は必ず秘書を通す。部署が直接CEOから指示を受けない
- 各部署は自分のCLAUDE.mdのルールに従って動く
- 判断に迷ったらCEOに確認を取る
- 記録はすべてObsidian Vaultに保存する
- 部署は必要になったら作る。いらない部署は作らない

## Obsidian Vaultとの関係

ひかるカンパニーのフォルダはObsidian Vault内に存在する。
既存の知識・資産は以下のフォルダにある：

- 10_自分OS/ ← コアテーマ軸・自分取扱説明書
- 20_知識ベース/ ← Web新着digest・各分野の知識
- 30_記事製造ライン/ ← ネタ・アイデア・ビジネス構想
- 40_NurseX戦略/ ← ブログ戦略・テンプレート
- 60_日報/ ← 日報
- 秘書メモ/ ← 秘書の記録
