# 📋 デイリーオペレーション

> ひかるが毎日この会社をどう動かすかの手順書。
> CEOがやることは「確認して承認する」だけ。Claudianが自動で動いている。

---

## 🤖 Claudianが毎朝自動でやること（指示不要）

| 時刻 | タスク | 保存先 |
|---|---|---|
| **7:06** | X投稿案を3つ生成（曜日テーマ別） | Obsidian / 日報フォルダ |
| **8:01** | digest処理（6軸×Web最新情報を検索・保存） | `20_知識ベース/Web新着digest/` |

**→ 朝起きたら両方完成している。**

---

## ☀️ 朝のルーティン（毎日・10分）

```
① X投稿案を確認 → 1つ選んで投稿する（5分）
② digestの結果を確認 → 気になるネタをメモする（5分）
```

**起動コマンド（必要な場合のみ）**
- 「今日のX投稿確認して」→ 生成済みの案を表示
- 「digestまとめて」→ 手動でdigest処理（自動分が遅れた場合）

---

## 🌙 夜のルーティン（帰宅後・曜日別）

| 曜日 | 夜の作業（60分） | 起動コマンド |
|---|---|---|
| **月** | ブログ記事制作① | 「ブログ記事書いて」 |
| **火** | ブログ記事制作② | 「ブログ記事書いて」 |
| **水** | note下書き or 既存ノート追記 | 「note記事書いて」 |
| **木**（休日） | 作戦会議 or リアル記事①制作 | 「作戦会議を開きます」 |
| **金** | リライト実行（週1本） | 「ブログ記事リライトして」 |
| **土** | 来週の素材ストック確認 | 「進捗は？」 |
| **日**（休日） | リアル記事②公開 + 週次レビュー | 「note公開する」→「今週の振り返りして」 |

## 📊 週次アウトプット目標

|---|---|---|

## 💬 CEOの口癖チートシート

| 言うだけでOK | 秘書が動く先 |
|---|---|
| 「おはよう、今日何やる？」 | 秘書 → ToDo確認 |
| 「digestまとめて」 | 秘書 → リサーチ部（手動実行） |
| 「X投稿作って」 | 秘書 → マーケティング部（手動実行） |
| 「ブログ記事書いて」 | 秘書 → ブログ事業部 |
| 「note記事書いて」 | 秘書 → note編集部 |
| 「リアル記事書いて」 | 秘書 → note編集部（note-real-article） |
| 「進捗は？」 | 秘書 → PM |
| 「日報作って」 | 秘書 → daily-report |
| 「セッション終了」 | 秘書 → session-end-obsidian |
| 「作戦会議を開きます」 | 秘書（作戦会議モード） |
| 「思いついたんだけど」 | 秘書（メモ → 分類 → 保存） |

## 注意事項

- ひかるは秘書にだけ話しかける。部署に直接指示しない
- 最終レビュー（記事の確認・投稿の承認）はCEOが必ず行う
- 「やらなきゃ」と思ったことは全部秘書に投げる。頭に残さない
- 夜勤明けは無理しない。X投稿確認だけでOK
- 会社の記録はすべてObsidianに残る。忘れても大丈夫
